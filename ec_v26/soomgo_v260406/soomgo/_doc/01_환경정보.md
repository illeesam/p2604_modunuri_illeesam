## 환경 변수

현재 버전은 백엔드 없이 목 데이터(Mock Data)로 동작하므로 별도의 환경 변수가 필요 없습니다.  
추후 API 연동 시 프로젝트 루트에 `.env` 파일을 생성하여 사용하세요.

```
VITE_API_BASE_URL=https://api.example.com
```

> Vite 환경 변수는 반드시 `VITE_` 접두사를 사용해야 클라이언트에서 접근 가능합니다.
