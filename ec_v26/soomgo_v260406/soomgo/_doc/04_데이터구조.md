# 데이터 구조 (Data Structure)

현재 백엔드 없이 `src/data/` 폴더의 목 데이터(Mock Data)로 동작합니다.

---

## categories.js

서비스 카테고리 10개와 인기 카테고리 그리드 데이터입니다.

```js
// 카테고리 객체 구조
{
  id: 'cleaning',          // URL 파라미터로 사용되는 고유 ID
  name: '청소',             // 표시 이름
  icon: '🧹',              // 이모지 아이콘
  color: '#88D8B0',        // 대표 색상 (HEX)
  preset: '아토피 케어 청소', // 홈 프리셋 카드 부제목
  services: ['거주 청소', '이사/입주 청소', ...], // 세부 서비스 목록
  img: 'https://...',      // 카테고리 대표 이미지 URL
}
```

## pros.js

고수(전문가) 데이터입니다. 현재 10명의 목 데이터가 있습니다.

```js
// 고수 객체 구조
{
  id: 1,                      // 고유 번호 (ProDetail URL 파라미터)
  name: '김민준',              // 이름
  category: 'cleaning',       // 소속 카테고리 ID
  service: '거주 청소',        // 주요 서비스명
  rating: 4.9,                // 평점 (0.0 ~ 5.0)
  reviewCount: 347,           // 리뷰 수
  responseRate: 98,           // 응답률 (%)
  avgPrice: '85,000원~',      // 평균 가격 표시
  location: '강남구',          // 활동 지역
  career: '경력 8년',          // 경력
  tags: ['당일예약가능', ...], // 태그 (최대 3개 표시)
  bio: '자기 소개 텍스트',      // 소개글
  img: 'https://...',         // 프로필 이미지 URL
  portfolio: ['https://...'], // 포트폴리오 이미지 URL 배열
  certifications: ['자격증명'], // 자격증/경력 목록
  services: [                 // 제공 서비스 및 가격
    { name: '원룸 청소', price: '50,000원~' }
  ]
}
```

## reviews.js

리뷰 데이터입니다.

```js
// 리뷰 객체 구조
{
  id: 1,
  proId: 1,             // 해당 고수의 ID (pros.id 참조)
  author: '김**',       // 작성자 (마스킹 처리)
  rating: 5,            // 별점 (1 ~ 5)
  date: '2025-03-15',   // 작성 날짜
  content: '리뷰 내용', // 리뷰 본문
  img: 'https://...',   // 첨부 이미지 URL (없으면 null)
}
```

## banners.js

메인 홈 배너 슬라이더 데이터입니다.

```js
// 배너 객체 구조
{
  id: 1,
  title: '새봄맞이 대청소',         // 배너 제목

  link: '/category/cleaning',       // 버튼 클릭 시 이동 경로
  bg: 'from-blue-500 to-cyan-400',  // Tailwind 그라디언트 클래스
  img: 'https://...',               // 배경 이미지 URL
}
```

## 데이터 확장 방법

### 고수 추가

`src/data/pros.js` 배열에 새 객체를 추가합니다. `id`는 기존 최댓값 + 1로 지정하세요.

### 카테고리 추가

`src/data/categories.js`의 `categories` 배열에 추가하고, `App.jsx` 라우팅은 이미 동적(`:id`)으로 처리되어 별도 수정이 필요 없습니다.

## 추후 API 연동 시

각 `src/data/*.js` 파일을 `useEffect` + `fetch` 또는 `axios` 호출로 교체하면 됩니다.

```js
// 예시: pros.js 목 데이터 → API 교체
const [pros, setPros] = useState([])

useEffect(() => {
  fetch(`${import.meta.env.VITE_API_BASE_URL}/pros`)
    .then(res => res.json())
    .then(data => setPros(data))
}, [])
```
