# 기타 사항 (Notes & References)

## 디자인 시스템

### 색상

| 이름 | HEX | 용도 |
|------|-----|------|
| Primary | `#1352F1` | 브랜드 메인 컬러, 버튼, 링크 |
| Primary Dark | `#0e3db5` | 호버 상태 |
| Primary Light | `#e8eeff` | 배경 강조, 태그 배경 |
| Secondary | `#FF6B35` | 강조 오렌지 |
| Background | `#F8F9FA` | 페이지 배경 |

### 폰트

- **Noto Sans KR** (Google Fonts)
- 굵기: 300 / 400 / 500 / 700 / 900

### 공통 컴포넌트 패턴

```
카드         rounded-2xl shadow-sm hover:shadow-md
버튼(Primary) bg-primary text-white rounded-full font-bold
버튼(아웃라인) border border-primary text-primary rounded-full
태그          bg-primary-light text-primary rounded-full text-xs
입력창        border border-gray-200 rounded-xl focus:border-primary
```

## 알려진 제한 사항

| 항목 | 현황 | 비고 |
|------|------|------|
| 인증 | UI만 구현 | 실제 로그인/회원가입 기능 없음 |
| 결제 | 미구현 | 마켓 장바구니 UI만 있음 |
| 실시간 알림 | 미구현 | 벨 아이콘 UI만 있음 |
| 지도 연동 | 미구현 | 위치 텍스트 입력만 지원 |
| 이미지 업로드 | 미구현 | Unsplash 이미지로 대체 |
| 채팅 | 미구현 | 메시지 버튼 UI만 있음 |

## Git 커밋 메시지 컨벤션 (권장)

```
feat:     새로운 기능 추가
fix:      버그 수정
style:    스타일/UI 변경
refactor: 코드 리팩토링
docs:     문서 수정
chore:    빌드, 패키지 등 기타 변경
```

예시:
```
feat: 고수 상세 페이지 포트폴리오 갤러리 추가
fix: 모바일 헤더 메뉴 닫힘 버그 수정
style: 카드 호버 효과 개선
```
