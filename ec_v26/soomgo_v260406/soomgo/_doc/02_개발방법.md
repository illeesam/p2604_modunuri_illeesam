# 개발 방법 (Development Guide)

## 1. 시작하기

### 저장소 클론 (이미 있는 경우 생략)

```bash
git clone <repository-url>
cd soomgo
```

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

---

## 2. 주요 명령어

| 명령어 | 설명 |
|--------|------|
| `npm run dev` | 개발 서버 시작 (HMR 지원) |
| `npm run build` | 프로덕션 빌드 (`dist/` 폴더 생성) |
| `npm run preview` | 빌드 결과물 로컬 미리보기 |

---

## 3. 프로젝트 구조

```
soomgo/
├── _doc/                   # 프로젝트 문서
├── public/                 # 정적 파일 (favicon 등)
├── src/
│   ├── components/         # 재사용 컴포넌트
│   │   ├── layout/         # Header, Footer
│   │   ├── home/           # 홈 페이지 섹션 컴포넌트
│   │   ├── pro/            # 고수 관련 컴포넌트
│   │   ├── request/        # 견적 요청 컴포넌트
│   │   └── common/         # 공통 UI 컴포넌트
│   ├── pages/              # 페이지 단위 컴포넌트
│   ├── data/               # 목 데이터 (Mock Data)
│   ├── App.jsx             # 라우터 설정
│   ├── main.jsx            # 앱 진입점
│   └── index.css           # 전역 스타일
├── tailwind.config.js      # Tailwind 설정
├── vite.config.js          # Vite 설정
└── package.json
```

---

## 4. 새 페이지 추가 방법

1. `src/pages/` 에 새 컴포넌트 파일 생성
2. `src/App.jsx` 에 `<Route>` 추가

```jsx
// src/App.jsx
import NewPage from './pages/NewPage'

<Route path="/new-page" element={<NewPage />} />
```

## 6. 목 데이터 수정 방법

`src/data/` 폴더 내 파일을 직접 수정합니다.

| 파일 | 내용 |
|------|------|
| `categories.js` | 서비스 카테고리 10개 |
| `pros.js` | 고수(전문가) 데이터 |
| `reviews.js` | 리뷰 데이터 |
| `banners.js` | 메인 배너 데이터 |

---

## 7. 스타일 가이드

Tailwind CSS 유틸리티 클래스를 사용합니다.  
커스텀 색상은 `tailwind.config.js` 에 정의되어 있습니다.

```js
// 사용 가능한 커스텀 색상
bg-primary        // #1352F1 (숨고 파란색)
bg-primary-dark   // #0e3db5
bg-primary-light  // #e8eeff
bg-secondary      // #FF6B35 (강조 오렌지)
bg-bg-main        // #F8F9FA
```

### 컴포넌트 스타일 패턴

```jsx
// 카드
<div className="bg-white rounded-2xl shadow-sm hover:shadow-md transition-shadow p-4">

// 버튼 (Primary)
<button className="bg-primary text-white px-4 py-2 rounded-full font-bold hover:bg-primary-dark transition-colors">

// 버튼 (Outline)
<button className="border border-primary text-primary px-4 py-2 rounded-full font-bold hover:bg-primary-light transition-colors">

// 태그/뱃지
<span className="bg-primary-light text-primary text-xs px-2 py-0.5 rounded-full">태그</span>
```
