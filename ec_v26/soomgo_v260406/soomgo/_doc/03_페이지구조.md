# 페이지 구조 (Page Structure)

## 라우팅 맵

| URL | 파일 | 설명 |
|-----|------|------|
| `/` | `pages/Home.jsx` | 메인 홈 |
| `/search?q=키워드` | `pages/Search.jsx` | 검색 결과 |
| `/category/:id` | `pages/Category.jsx` | 카테고리별 고수 목록 |
| `/pro/:id` | `pages/ProDetail.jsx` | 고수 상세 프로필 |
| `/request` | `pages/Request.jsx` | 견적 요청 폼 |
| `/login` | `pages/Login.jsx` | 로그인 |
| `/signup` | `pages/Signup.jsx` | 회원가입 |
| `/community` | `pages/Community.jsx` | 커뮤니티 |
| `/market` | `pages/Market.jsx` | 마켓 |

---

## 홈 페이지 섹션 구성

```
Home.jsx
├── HeroBanner          배너 슬라이더 (자동재생, 인디케이터)
├── PresetServices      "지금 필요한 서비스" 10개 카드
├── PopularCategories   인기 카테고리 아이콘 그리드
├── RecommendedPros     "오늘의 추천 고수" 카드 목록
├── NearbyContent       "강남구 주변에서 많이 본" 콘텐츠
├── CommunityPreview    커뮤니티 인기글 미리보기
└── App Download Banner 앱 다운로드 배너
```

## 주요 컴포넌트 흐름

```
App.jsx (라우터)
│
├── components/layout/Header.jsx  ← 모든 페이지 공통
│   ├── 로고 → /
│   ├── 검색바 → /search?q=
│   ├── 카테고리 네비 → /category/:id
│   └── 로그인/회원가입 → /login, /signup
│
├── [각 페이지 컴포넌트]
│
└── components/layout/Footer.jsx  ← 모든 페이지 공통
```

---

## 고수 상세 페이지 탭 구조

```
ProDetail.jsx (/pro/:id)
├── 탭: 소개       ← 자기소개, 자격증/경력
├── 탭: 포트폴리오  ← 이미지 갤러리
├── 탭: 서비스/가격 ← 서비스 목록 + 가격
└── 탭: 리뷰       ← 리뷰 목록 (ReviewCard)
```

---

## 견적 요청 폼 단계

```
Request.jsx (/request)
├── Step 1: 서비스 선택   ← 카테고리 + 세부 서비스
├── Step 2: 상세 정보     ← 요청 내용 textarea
├── Step 3: 일정/위치     ← 날짜, 위치, 이름, 연락처
└── Step 4: 완료          ← 요청 완료 화면
```
