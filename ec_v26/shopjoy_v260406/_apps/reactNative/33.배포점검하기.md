## E. 사고 대응

### E-1. 단계적 출시 일시정지

#### Android
Play Console → Production → **출시 중단** (Halt rollout).

#### iOS
App Store Connect → 단계별 출시 → **Pause**.

### E-2. 핫픽스 출시

```bash
# 1. 코드 수정
# 2. 버전 증가
# 3. 태그
git tag rn-fo-v1.0.1
git push origin rn-fo-v1.0.1

# 4. Codemagic 자동 빌드
# 5. Internal / TestFlight 검증
# 6. Production 5% → 점진 확대
```

### E-3. CodePush (OTA, 선택)

JS 번들만 핫 업데이트 (네이티브 코드 변경 X):

```bash
npm install --save react-native-code-push
```

App Center 또는 자체 서버에 번들 호스팅 → 앱이 시작 시 자동 다운로드.

> ⚠️ Apple 정책: 결제 / 핵심 기능 변경은 OTA 금지. 버그픽스 / 문구 수정 위주.
