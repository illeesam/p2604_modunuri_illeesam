## A. 일상 개발 워크플로우

### A-1. 두 개의 터미널

**Terminal 1 — Metro Bundler**:
```bash
cd _apps/reactNative/ShopjoyFoApp
npm start
```

**Terminal 2 — 빌드 + 실행**:
```bash
npm run android:local      # 또는 ios:local
```

### A-2. VS Code Live Server (선택, WebView 콘텐츠 라이브 리로드용)

별도 창: ShopJoy 웹 프로젝트 → `index.html` 우클릭 → Open with Live Server.

---

## B. Android Studio 사용

### B-1. 프로젝트 열기

```bash
open -a "Android Studio" _apps/reactNative/ShopjoyFoApp/android
```

또는 Android Studio → Open → `android/` 폴더 선택.

### B-2. 환경별 Build Variant 선택

좌하단 **Build Variants** 탭 → Active Build Variant:
- `localDebug` / `devDebug` / `prodDebug`
- `localRelease` / `devRelease` / `prodRelease`

> RN 의 productFlavors 설정은 `android/app/build.gradle` 에서 직접 추가 필요.

### B-3. ▶ 실행

좌상단 디바이스 → AVD 선택 → ▶.

---

## C. Xcode 사용 (macOS)

### C-1. Workspace 열기

```bash
open _apps/reactNative/ShopjoyFoApp/ios/ShopjoyFoApp.xcworkspace
```

> **`.xcodeproj` 가 아닌 `.xcworkspace`** 를 열어야 CocoaPods 의존성 인식.

### C-2. Scheme 선택

좌상단 Scheme 드롭다운 → `ShopjoyFoApp-Local` / `Dev` / `Prod`.

> Scheme 이 없으면 RN init 후 수동 추가 필요. **Manage Schemes** 에서 Duplicate.

### C-3. ▶

디바이스 선택 → ▶ (Cmd+R).

## D. RN Dev Menu 활용

핸드폰에서 흔들기 (또는 Cmd+D iOS Sim, Cmd+M Android Emu):

| 메뉴 | 동작 |
|---|---|
| Reload | JS 번들 다시 로드 |
| Debug with Chrome | Chrome DevTools 로 디버깅 |
| Show Inspector | UI 요소 검사 |
| Toggle Performance Monitor | FPS / RAM 표시 |

## E. 환경 전환

### E-1. local → dev

```bash
npm run android:dev
# 또는
npm run ios:dev
```

`.env.dev` 파일 자동 로드 → Netlify 개발 URL 가리킴.

### E-2. local → prod

```bash
npm run android:prod
# 또는
npm run ios:prod
```

## F. 환경별 applicationId / Bundle ID 분리 (선택)

### F-1. Android (build.gradle)

`android/app/build.gradle` 에 추가:

```gradle
android {
    flavorDimensions "env"
    productFlavors {
        local {
            dimension "env"
            applicationIdSuffix ".local"
            resValue "string", "app_name", "ShopJoy FO Local"
        }
        dev {
            dimension "env"
            applicationIdSuffix ".dev"
            resValue "string", "app_name", "ShopJoy FO Dev"
        }
        prod {
            dimension "env"
            resValue "string", "app_name", "ShopJoy"
        }
    }
}
```

### F-2. iOS (Xcode)

Xcode → Project → Configuration:
- Debug → Debug-Local / Debug-Dev / Debug-Prod
- 각 Configuration 별 `PRODUCT_BUNDLE_IDENTIFIER` 변경

## G. 단축키

- Shift+F9 — Debug

### Xcode
- Cmd+R — Run
- Cmd+B — Build
- Cmd+. — Stop
- Cmd+Shift+K — Clean

## H. 트러블슈팅

| 증상 | 해결 |
|---|---|
| Metro 가 멈춤 | `Ctrl+C` 후 `npx react-native start --reset-cache` |
| `Unable to load script` | Metro 가 안 켜져 있거나 다른 포트 |
| Android 빌드 OOM | `android/gradle.properties` 의 `org.gradle.jvmargs=-Xmx4096m` |
| iOS Pod 충돌 | `cd ios && pod deintegrate && pod install` |
| Fast Refresh 안 됨 | Dev Menu → Disable Fast Refresh → Enable Fast Refresh |
