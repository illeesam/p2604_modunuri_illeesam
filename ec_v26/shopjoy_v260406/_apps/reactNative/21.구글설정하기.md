# 21. Google (Firebase) 설정하기 (RN) — 따라하기

ShopJoy RN 앱에 FCM 푸시 추가.

## 사전 준비

- [ ] [`11.개발환경.md`](11.개발환경.md) 완료
- [ ] Google 계정

## 1단계: Firebase 프로젝트 + Android 앱 등록

### 1-1. 프로젝트 생성

https://console.firebase.google.com → 프로젝트 추가 → 이름 `shopjoy-prod`.

### 1-2. Android 앱 추가

각 환경별:

| 환경 | 패키지명 |
|---|---|
| FO local | `com.shopjoy.fo.local` |
| FO dev | `com.shopjoy.fo.dev` |
| FO prod | `com.shopjoy.fo` |
| BO local/dev/prod | `com.shopjoy.bo.*` |

→ google-services.json 다운로드.

### 1-3. 파일 배치

```bash
mv ~/Downloads/google-services.json \
   _apps/reactNative/ShopjoyFoApp/android/app/google-services.json
```

---

## 2단계: Android Gradle 수정

### 2-1. android/build.gradle

```gradle
buildscript {
    dependencies {
        classpath("com.google.gms:google-services:4.4.2")
    }
}
```

### 2-2. android/app/build.gradle

상단:
```gradle
apply plugin: 'com.google.gms.google-services'
```

하단:
```gradle
dependencies {
    implementation platform('com.google.firebase:firebase-bom:33.4.0')
    implementation 'com.google.firebase:firebase-messaging'
}
```

## 3단계: iOS 앱 등록 (macOS)

### 3-1. Firebase Console → iOS 추가 → GoogleService-Info.plist 다운로드

### 3-2. 파일 배치 + Xcode 등록

```bash
cd _apps/reactNative/ShopjoyFoApp
mv ~/Downloads/GoogleService-Info.plist ios/ShopjoyFoApp/
open ios/ShopjoyFoApp.xcworkspace
```

Xcode → 좌측 우클릭 → Add Files → `GoogleService-Info.plist` 선택 → Copy items if needed ✅ → Add to targets ✅.

### 3-3. APNs 인증 키

[`_apps/ios/22.IOS설정하기.md`](../ios/22.IOS설정하기.md) 의 **2단계** 따라 `.p8` 발급 → Firebase Console → 클라우드 메시징 → APNs 인증 키 업로드.

### 3-4. AppDelegate 수정

`ios/ShopjoyFoApp/AppDelegate.mm`:

```objc
#import <Firebase.h>

- (BOOL)application:(UIApplication *)application
    didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
  [FIRApp configure];
  // ... 기존 코드 ...
}
```

### 3-5. Pod 추가

`ios/Podfile`:
```ruby
target 'ShopjoyFoApp' do
  pod 'Firebase/Core'
  pod 'Firebase/Messaging'
end
```

```bash
cd ios && pod install && cd ..
```

## 4단계: react-native-firebase 사용 (이미 package.json 에 포함)

`@react-native-firebase/app` + `@react-native-firebase/messaging` 가 이미 설치됨.

### 4-1. App.tsx 에 푸시 코드 추가

```typescript
import messaging from '@react-native-firebase/messaging';
import {PermissionsAndroid, Platform} from 'react-native';

useEffect(() => {
  async function setupPush() {
    // Android 13+ 권한
    if (Platform.OS === 'android' && Number(Platform.Version) >= 33) {
      await PermissionsAndroid.request(
        PermissionsAndroid.PERMISSIONS.POST_NOTIFICATIONS
      );
    }

    // iOS 권한
    if (Platform.OS === 'ios') {
      const auth = await messaging().requestPermission();
      if (auth !== messaging.AuthorizationStatus.AUTHORIZED &&
          auth !== messaging.AuthorizationStatus.PROVISIONAL) {
        return;
      }
    }

    const token = await messaging().getToken();
    console.log('[FCM] token:', token);
    // TODO: 백엔드 등록 — POST /fo/my/device-token

    // 포그라운드 메시지
    messaging().onMessage(async remoteMessage => {
      console.log('[FCM] foreground:', remoteMessage);
    });

    // 백그라운드/종료 상태에서 푸시 탭
    messaging().onNotificationOpenedApp(remoteMessage => {
      const url = remoteMessage.data?.url;
      if (url) {
        webRef.current?.injectJavaScript(
          `if(window.onAppDeepLink) window.onAppDeepLink('${url}');`
        );
      }
    });
  }
  setupPush();
}, []);
```

## 5단계: 테스트 발송

Firebase Console → Cloud Messaging → 새 캠페인 → 알림 → 테스트 메시지 → FCM 토큰 입력.

핸드폰에 푸시 도착 확인.

## 트러블슈팅

| 증상 | 해결 |
|---|---|
| Android: `Default FirebaseApp not initialized` | google-services 플러그인 미적용 / json 경로 |
| iOS: 토큰 nil | 시뮬레이터 X (실기기 필수), APNs 키 업로드 확인 |
| iOS Build error: `module 'Firebase' not found` | `pod install` 다시 + `.xcworkspace` 사용 |
| 권한 다이얼로그 안 뜸 | iOS Info.plist NSUserNotificationsUsageDescription 추가 |
