# 20. Apple ID 가입 + iOS 인증번호 받기 — 따라하기 (React Native)

> RN iOS 빌드도 결국 Xcode 워크스페이스 (`ios/ShopjoyFoApp.xcworkspace`) 라서 절차가 순수 iOS 와 동일합니다.
> 자세한 절차는 → [`_apps/ios/20.IOS계정가입및IOS인증번호받기.md`](../ios/20.IOS계정가입및IOS인증번호받기.md)

## iOS 인증번호 — 한눈에

| 용도 | iOS 식별자 |
|---|---|
| 앱 식별 | **Bundle ID** (RN `PRODUCT_BUNDLE_IDENTIFIER`) |
| 팀 식별 | **Team ID** (10자) |
| 푸시 발송 | **APNs 인증 키 (.p8)** + Key ID |
| 코드 서명 | **Distribution Certificate** |
| 빌드 권한 | **Provisioning Profile** |

## 사전 준비

- [ ] **macOS 14+** + Xcode 15.4+
- [ ] CocoaPods 설치 (`sudo gem install cocoapods`)
- [ ] RN 0.75+
- [ ] 신용카드 (Apple Developer Program $99/년)

---

## 1단계: Apple ID 가입

https://appleid.apple.com → **Apple ID 생성** → 가입.

**2단계 인증 필수**.

---

## 2단계: Apple Developer Program 가입

https://developer.apple.com/programs/enroll → **Individual** / **Organization** → 결제 ($99) → 1~2일 승인.

---

## 3단계: Team ID 확인

Apple Developer Portal → **Membership** → Team ID (10자) 메모.

---

## 4단계: Bundle ID 등록

### 4-0. RN Bundle ID 변경

RN 의 기본 Bundle ID 는 `init` 시 자동 생성됨. 변경:

```bash
cd _apps/reactNative/ShopjoyFoApp
npx react-native-rename "ShopjoyFoApp" -b com.shopjoy.fo
```

또는 Xcode 에서 직접:

```bash
open ios/ShopjoyFoApp.xcworkspace
```

ShopjoyFoApp 타겟 → **General → Bundle Identifier** → `com.shopjoy.fo`.

### 4-1. Apple Developer Portal 등록

1. https://developer.apple.com/account → **Identifiers** → **+**
2. **App IDs** → **App** → **Continue**
3. Bundle ID: `com.shopjoy.fo` (Explicit)
4. Capabilities ✅ **Push Notifications**
5. **Continue → Register**

### 4-2. 환경별 Bundle ID 분리 (선택)

RN 도 Xcode Configuration 별로 Bundle ID 분리 가능:

| Configuration | Bundle ID |
|---|---|
| Debug-Local / Release-Local | `com.shopjoy.fo.local` |
| Debug-Dev / Release-Dev | `com.shopjoy.fo.dev` |
| Debug-Prod / Release-Prod | `com.shopjoy.fo` |

각 환경별 Bundle ID 모두 Apple Developer Portal 에 등록.

## 5단계: APNs 인증 키 (.p8) 발급

`@react-native-firebase/messaging` 가 사용.

1. https://developer.apple.com/account → **Keys** → **+**
2. Key Name: `ShopJoy APNs Key`
3. ✅ **Apple Push Notifications service (APNs)** → **Configure** → **All Apps** → **Save**
4. **Continue → Register → Download** → `AuthKey_XXXX.p8` (한 번만)
5. **Key ID** + **Team ID** 메모

## 6단계: Distribution Certificate + Provisioning Profile

상세 → [`_apps/ios/20.IOS계정가입및IOS인증번호받기.md`](../ios/20.IOS계정가입및IOS인증번호받기.md) 의 **6/7단계**.

요약:
1. Mac 키체인 → CSR 생성
2. Apple Developer **Certificates → +** → **Apple Distribution** → CSR 업로드 → 다운로드 → 키체인 등록
3. **Profiles → +** → **App Store** → Bundle ID + Certificate → 다운로드 → 더블클릭 → Xcode 자동 등록

## 7단계: RN iOS Pod + Xcode 자동 서명

```bash
cd _apps/reactNative/ShopjoyFoApp
npm install
cd ios

cd ..
open ios/ShopjoyFoApp.xcworkspace    # ⚠️ .xcworkspace
```

Xcode → **ShopjoyFoApp** 타겟 → **Signing & Capabilities**:
- ✅ Automatically manage signing
- Team 선택
- Bundle Identifier: `com.shopjoy.fo`

**+ Capability**:
- ✅ Push Notifications
- ✅ Background Modes → Remote notifications

## 8단계: Firebase 등록 + GoogleService-Info.plist

### 8-1. Firebase Console iOS 앱 추가

https://console.firebase.google.com → 프로젝트 → ⚙ → **앱 추가** → **iOS** → Bundle ID: `com.shopjoy.fo` → 등록 → `GoogleService-Info.plist` 다운로드.

### 8-2. 파일 배치

```bash
mv ~/Downloads/GoogleService-Info.plist \
   _apps/reactNative/ShopjoyFoApp/ios/ShopjoyFoApp/GoogleService-Info.plist
```

### 8-3. Xcode 에 등록

`open ios/ShopjoyFoApp.xcworkspace` → 좌측 ShopjoyFoApp 폴더 우클릭 → **Add Files to "ShopjoyFoApp"** → `GoogleService-Info.plist` 선택 → **Copy items if needed** ✅ → **Add to targets: ShopjoyFoApp** ✅.

### 8-4. AppDelegate 수정

`ios/ShopjoyFoApp/AppDelegate.mm` (또는 `.swift` ):

```objc
#import <Firebase.h>

- (BOOL)application:(UIApplication *)application
    didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
  [FIRApp configure];
  // 기존 RN 초기화 코드...
}
```

### 8-5. Podfile 추가

`ios/Podfile`:

```ruby
target 'ShopjoyFoApp' do
  pod 'Firebase/Core'
  pod 'Firebase/Messaging'
end
```

```bash
cd ios && pod install && cd ..
```

## 11단계: 카카오 SDK iOS (사용 시)

`@react-native-seoul/kakao-login` 등 사용 시:

1. https://developers.kakao.com → 내 애플리케이션 → **플랫폼 → iOS 플랫폼 등록**

## 트러블슈팅

| 증상 | 해결 |
|---|---|
| `pod install` arm64 오류 (M1/M2) | `arch -x86_64 pod install` 또는 CocoaPods 1.15+ |
| `module 'Firebase' not found` | `cd ios && pod deintegrate && pod install` |
| `Code signing required` | Step 7 의 Team / Bundle ID 재확인 |
| 시뮬레이터에서 토큰 nil | 실기기 필수 |
| `FirebaseApp not configured` | AppDelegate 의 `[FIRApp configure]` 추가 누락 |
| Xcode 가 `.xcodeproj` 만 인식 | `.xcworkspace` 로 열기 |
| RN 0.75+ 에서 `firebase` 호환성 | `@react-native-firebase` 22+ 사용 |
