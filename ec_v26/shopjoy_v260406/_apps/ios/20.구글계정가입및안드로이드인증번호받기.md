# 20. Google 계정 가입 + Android 인증번호 받기 — 따라하기 (iOS 폴더 — 참고)

> 이 문서는 **iOS Native 앱** 빌드와는 직접 관련이 없습니다. iOS 에서는 SHA-1 대신 **APNs 인증 키 (.p8) + Apple 개발자 인증서**를 사용합니다.
> 다만, **Firebase / Google Cloud Console 사용 시 Google 계정 가입은 공통으로 필요**하므로 이 문서를 유지합니다.

## 사전 준비

- [ ] 인터넷 연결됨

---

## 1단계: Google 계정 만들기

### 1-1. 가입

브라우저에서 https://accounts.google.com/signup 접속:

| 항목 | 입력값 |
|---|---|
| 이름 | 본인 이름 |
| 사용자 이름 | `shopjoy.dev@gmail.com` |
| 비밀번호 | 강력한 비밀번호 (12자+) |

→ **다음** → 전화번호 SMS 인증 → 복구 정보 → 약관 동의 → 가입 완료.

### 1-2. 2단계 인증 활성화

https://myaccount.google.com/security → **2단계 인증 → 사용**.

> ⚠️ **운영 키 / Firebase 관리자 계정은 반드시 2단계 인증 필수**.

### 1-3. 운영 / 개인 계정 분리

| 용도 | 권장 |
|---|---|
| Firebase / Google Cloud / Google Analytics | `shopjoy.ops@gmail.com` |
| 본인 일상 | `myname@gmail.com` |

## 2단계: iOS 에서는 SHA-1 대신 무엇을 사용하나?

| Android | iOS |
|---|---|
| keystore SHA-1 / SHA-256 | **APNs 인증 키 (.p8)** + **Bundle ID** |
| `~/.android/debug.keystore` | Apple Developer Portal Distribution Certificate |
| Play Console → 앱 서명 키 SHA | Apple Developer Portal Provisioning Profile |
| Kakao 키 해시 (SHA-1 base64) | Kakao iOS 플랫폼 등록은 **Bundle ID** 만 |

iOS 인증서 / 키 발급은 [`22.IOS설정하기.md`](22.IOS설정하기.md) 참조.

## 3단계: iOS / Android 양쪽 사용 시 (Capacitor / RN / Flutter)

ShopJoy 앱이 **iOS + Android 양쪽 출시** 라면:

### 3-1. Firebase 콘솔에 양쪽 등록

Firebase 한 프로젝트에 다음 4~6개 앱 모두 등록:

| 앱 | 플랫폼 | 식별자 |
|---|---|---|
| ShopJoy FO Android | Android | `com.shopjoy.fo` |
| ShopJoy FO iOS | iOS | `com.shopjoy.fo` (Bundle ID) |
| ShopJoy BO Android | Android | `com.shopjoy.bo` |
| ShopJoy BO iOS | iOS | `com.shopjoy.bo` |

### 3-2. Android 등록 시 SHA-1 / SHA-256 필요

Android 앱 등록 시 SHA 입력 — 본 문서의 다른 폴더 참조:

- Android Native: [`_apps/android/20.구글계정가입및안드로이드인증번호받기.md`](../android/20.구글계정가입및안드로이드인증번호받기.md)
- Capacitor: [`_apps/capacitor/20.구글계정가입및안드로이드인증번호받기.md`](../capacitor/20.구글계정가입및안드로이드인증번호받기.md)
- Flutter: [`_apps/flutter/20.구글계정가입및안드로이드인증번호받기.md`](../flutter/20.구글계정가입및안드로이드인증번호받기.md)
- React Native: [`_apps/reactNative/20.구글계정가입및안드로이드인증번호받기.md`](../reactNative/20.구글계정가입및안드로이드인증번호받기.md)

### 3-3. iOS 등록 시 SHA 불필요

iOS 앱 등록 시 SHA 항목 없음. Bundle ID 만 입력. APNs 키는 별도로 [`22.IOS설정하기.md`](22.IOS설정하기.md) 따라 업로드.

---

## 4단계: Apple 계정도 함께 준비

iOS 빌드를 위해서는 Google 계정 외에 **Apple ID** + **Apple Developer Program 가입** 도 필요합니다:

| 계정 종류 | 가입 | 비용 |
|---|---|---|
| Apple ID | https://appleid.apple.com | 무료 |
| Apple Developer Program | https://developer.apple.com/programs | $99 / 년 |

상세 가입 절차 → [`22.IOS설정하기.md`](22.IOS설정하기.md).

## 트러블슈팅

| 증상 | 해결 |
|---|---|
| iOS 에서 `InvalidKeyHash` | iOS 는 키 해시 사용 안 함 → Bundle ID 확인 |
| Firebase iOS 토큰 미수신 | APNs 키 (.p8) 미업로드 → [`22.IOS설정하기.md`](22.IOS설정하기.md) |
| Apple Developer 가입 검증 1~2일 | 기본 절차. 카드 결제 후 대기 |
