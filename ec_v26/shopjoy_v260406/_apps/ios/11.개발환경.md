# 11. iOS 개발환경 구축 — 따라하기

이 문서를 처음부터 끝까지 따라하면 iOS 개발 환경이 완성됩니다.

> ⚠️ **macOS 가 반드시 필요합니다.** Windows / Linux 에서는 iOS 빌드가 불가능합니다.
> Mac 이 없다면 → [`33.배포점검하기.md`](33.배포점검하기.md) 의 클라우드 빌드 옵션 참조.

## 사전 준비 체크리스트

이 문서를 시작하기 전에 다음을 확인하세요:

- [ ] macOS 14 (Sonoma) 이상이 설치되어 있다
- [ ] 디스크 여유공간 50 GB 이상
- [ ] Apple ID 가 있다 (없으면 https://appleid.apple.com 에서 생성)
- [ ] 인터넷 연결됨

---

## 1단계: Xcode 설치

### 1-1. App Store 에서 Xcode 검색 → 설치

1. macOS Dock → **App Store** 아이콘 클릭
2. 좌측 상단 검색창에 `Xcode` 입력
3. **Xcode** (Apple 제공) → **받기** 또는 **설치** 클릭
4. Apple ID 로 인증
5. 다운로드 완료 대기 (약 15~30분, 10 GB)

### 1-2. Xcode 첫 실행

1. **Launchpad → Xcode** 실행
2. 라이선스 동의 → **Agree**
3. **Install** 버튼이 나타나면 클릭 → 추가 컴포넌트 자동 설치 (5~10분)

### 1-3. 설치 확인

터미널 열기 (Cmd+Space → "Terminal" → Enter):

```bash
xcodebuild -version
```

다음과 같이 보이면 OK:

```
Xcode 15.4
Build version 15F31d
```

---

## 2단계: Xcode Command Line Tools 설치

### 2-1. 설치 명령

```bash
xcode-select --install
```

다이얼로그가 뜨면 **Install** 클릭. 약 5분 대기.

### 2-2. 라이선스 동의

```bash
sudo xcodebuild -license accept
```

비밀번호 입력 → 라이선스 자동 동의.

### 2-3. 확인

```bash
xcode-select -p
```

다음과 같이 보이면 OK:

```
/Applications/Xcode.app/Contents/Developer
```

---

## 3단계: Homebrew 설치

### 3-1. 설치 명령

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

설치 완료 후, 터미널에 출력된 PATH 추가 명령을 그대로 복사해서 실행 (Apple Silicon 사용자만):

```bash
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
```

### 3-2. 확인

```bash
brew --version
```

다음과 같이 보이면 OK:

```
Homebrew 4.x.x
```

---

## 4단계: CocoaPods 설치

### 4-1. 설치 명령

```bash
sudo gem install cocoapods
```

비밀번호 입력 → 약 3분 대기.

### 4-2. 확인

```bash
pod --version
```

다음과 같이 보이면 OK:

```
1.15.x
```

---

## 5단계: XcodeGen 설치

ShopJoy iOS 프로젝트는 `.xcodeproj` 를 git 에 포함하지 않고 `project.yml` 에서 자동 생성합니다.

### 5-1. 설치 명령

```bash
brew install xcodegen
```

### 5-2. 확인

```bash
xcodegen --version
```

다음과 같이 보이면 OK:

```
Version: 2.42.0
```

---

## 6단계: Apple Developer 가입

### 6-1. 무료 계정 (Personal Team)

7일 동안 실기기 테스트 가능. App Store 배포 불가.

1. Xcode → **Settings** → **Accounts** 탭
2. 좌하단 **+** → **Apple ID**
3. 본인 Apple ID + 비밀번호 입력
4. 자동으로 **Personal Team** 표시됨 → 그대로 사용

### 6-2. 유료 계정 (App Store 배포용)

App Store 배포가 필요하면:

1. https://developer.apple.com/programs 접속
2. **Enroll** 클릭
3. 개인 또는 회사 선택
4. $99 결제
5. 1~2 영업일 검증 대기

---

## 7단계: ShopJoy iOS 프로젝트 열기

### 7-1. 프로젝트 디렉토리로 이동

```bash
cd _apps/ios/ShopjoyFoApp
```

### 7-2. .xcodeproj 자동 생성

```bash
xcodegen generate
```

다음과 같이 보이면 OK:

```
⚙️  Generating plists...
⚙️  Generating project...
⚙️  Writing project...
Created project at /...../ShopjoyFoApp.xcodeproj
```

### 7-3. Xcode 에서 열기

```bash
open ShopjoyFoApp.xcodeproj
```

Xcode 창이 열리고 좌측에 프로젝트 트리가 보이면 OK.

---

## 8단계: 시뮬레이터에서 빌드 검증

### 8-1. Scheme 확인

Xcode 창 좌상단 Scheme 드롭다운 → 다음 3개가 보이는지 확인:

- `ShopjoyFoApp-Local`
- `ShopjoyFoApp-Dev`
- `ShopjoyFoApp-Prod`

### 8-2. 시뮬레이터 선택

Xcode 창 좌상단 디바이스 드롭다운 → **iPhone 15 Pro** 선택.

### 8-3. 빌드 + 실행

상단 ▶ 버튼 클릭 (단축키 Cmd+R).

처음 실행 시 시뮬레이터 부팅에 1~2분 소요. ShopJoy 앱이 시뮬레이터에 표시되면 성공.

---

## 9단계: VS Code Live Server 연동 (local 환경 테스트)

`local` 스킴은 `http://127.0.0.1:5501` 을 가리킵니다. Mac 에서 ShopJoy 웹 서버를 띄워야 동작합니다.

### 9-1. VS Code 설치

```bash
brew install --cask visual-studio-code
```

### 9-2. Live Server 확장 설치

VS Code 실행 → 좌측 확장 아이콘 → `Live Server` 검색 → **Install**.

### 9-3. ShopJoy 웹 프로젝트 열기

VS Code 에서 `_pjt_github/p2604_modunuri_illeesam/ec_v26/shopjoy_v260406` 폴더 열기.

### 9-4. Live Server 시작

`index.html` 우클릭 → **Open with Live Server**.

브라우저가 자동으로 열리면 → 포트 5501 가동 중.

### 9-5. iOS 시뮬레이터에서 다시 ▶

ShopJoy 가 시뮬레이터에서 정상 표시되면 환경 구축 완료!
