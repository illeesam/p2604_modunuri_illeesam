### Crashlytics 추가

```yaml
dependencies:
  firebase_crashlytics: ^4.1.3
```

`main.dart`:
```dart
FlutterError.onError = FirebaseCrashlytics.instance.recordFlutterFatalError;

PlatformDispatcher.instance.onError = (error, stack) {
  FirebaseCrashlytics.instance.recordError(error, stack, fatal: true);
  return true;
};
```

### E-2. 핫픽스 출시

```bash

git push origin flutter-fo-v1.0.1

# 4. Codemagic 자동 빌드
# 5. Internal / TestFlight 검증
# 6. Production 5% → 점진 확대
```

### E-3. Shorebird (OTA, 선택)

Flutter 의 OTA 솔루션. Dart 코드만 핫 업데이트:

```bash
dart pub global activate shorebird_cli
shorebird init
shorebird release android --flavor prod --dart-define=APP_ENV=prod
shorebird patch android --flavor prod --dart-define=APP_ENV=prod
```

> ⚠️ 결제 / 핵심 기능 변경은 OTA 금지 (앱스토어 정책). 버그픽스 / 문구 수정 위주.
