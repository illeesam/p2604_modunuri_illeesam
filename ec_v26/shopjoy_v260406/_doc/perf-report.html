⚡ ShopJoy 로딩 성능 레포트

로컬 / HTTP/1.1(라이브) / HTTP/2(라이브) 3가지 환경 비교 측정 도구

Step 1 — 로컬 측정 (지금 이 환경)
브라우저에서 bo.html 열기 → 개발자도구(F12) → Network 탭 → 새로고침(Ctrl+Shift+R)
Network 탭 상단 Finish 시간과 총 요청 수를 아래에 입력하세요. 
확인 위치: Network 탭 하단 상태바
예) "156 requests | 2.1 MB transferred | Finish: 312 ms"
Step 2 — 라이브 HTTP/1.1 측정
라이브 서버에서 HTTP/2 비활성화 상태로 bo.html 열기
(또는 Chrome DevTools → Network 탭 → 우클릭 헤더 → Protocol 컬럼 추가 후 http/1.1 확인)
HTTP/1.1 강제: 브라우저 주소창에 http:// 로 접속하면 HTTP/1.1만 사용됨.
Step 3 — 라이브 HTTP/2 측정
HTTP/2 활성화 후 https:// 로 접속 → 동일하게 측정
Network 탭 Protocol 컬럼에서 h2 표시 확인 필수
Step 4 — 측정 환경 메모 (선택)
네트워크 환경, 서버 사양, 측정 조건 등 자유롭게 입력
