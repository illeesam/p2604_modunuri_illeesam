# 511. 프로모션 - 쿠폰 정책

## 목적
쿠폰 발행, 배포, 사용을 통한 판매 촉진 정책 정의

## 범위
- 쿠폰 생성 및 발행
- 쿠폰 할당 및 배포
- 쿠폰 사용 및 환불
- 쿠폰 상태 관리

## 쿠폰 상태 (COUPON_STATUS)
| 상태 | 코드 | 설명 |
|------|------|------|
| 활성 | ACTIVE | 사용 가능 상태 |
| 비활성 | INACTIVE | 사용 불가 상태 |
| 만료 | EXPIRED | 유효기간 만료 |

## 쿠폰 유형 (COUPON_TYPE)
| 유형 | 코드 | 설명 | 예시 |
|------|------|------|------|
| 정률 | RATE | 할인율 적용 | 20% 할인 |
| 정액 | FIXED | 고정금액 할인 | 10,000원 할인 |

## 쿠폰 대상 (COUPON_TARGET)
| 대상 | 코드 | 설명 |
|------|------|------|
| 전체 | ALL | 모든 회원 |
| 회원 | MEMBER | 특정 회원 그룹 |
| 등급 | GRADE | 특정 회원등급 (VIP, GOLD 등) |

## 주요 정책

### 1. 쿠폰 생성
- **필수정보**:
  - 쿠폰명 (100자 이내)
  - 쿠폰코드 (50자, UNIQUE)
  - 할인유형 (정률/정액)
  - 할인율/금액
  - 최소주문금액
- **선택정보**:
  - 최대할인한도
  - 발급한도
  - 유효기간

### 2. 쿠폰 발행 방식
- **일반발행**: 특정 기간 공개 발행
- **특정회원 배포**: 타겟 회원에게만 발급
  - 생일쿠폰, 등급별쿠폰, 휴면복구쿠폰 등
- **이벤트**: 이벤트 참여 시 쿠폰 지급
- **추천**: 친구 추천 보상 쿠폰

### 3. 쿠폰 적용 규칙
- **최소주문금액**: 
  - 기본 조건 필수
  - 예: 50,000원 이상
- **최대할인한도**: 
  - 정률쿠폰 시에만 적용
  - 예: 최대 20,000원까지만 할인
- **중복사용**: 1회 주문당 쿠폰 1개만 사용
- **선택적용**: 최적 할인을 자동으로 선택

### 4. 쿠폰 할인 한계
- **정률할인**: 최대 50% 이내
- **정액할인**: 최소 1,000원 ~ 최대 100,000원
- **중복할인**: 쿠폰 + 카테고리할인 + 상품할인은 1개만 적용
  - 최대 할인액 먼저 적용

### 5. 쿠폰 유효기간
- **기본기간**: 3개월
- **장기쿠폰**: 최대 1년
- **자동만료**: 유효기간 종료 시 자동 사용불가
- **만료알림**: 7일 전 회원에게 알림

### 6. 쿠폰 발급한도
- **총발급한도**: 최대 10,000개
- **회원당한도**: 기본 3개까지
  - 등급별 다르게 설정 가능
  - VIP: 5개까지
- **일일발급한도**: 최대 100개/일

### 7. 쿠폰 취소 및 환불
- **미사용 취소**: 자동 반납 (유효기간 만료)
- **사용후 취소**: 
  - 주문취소 시 쿠폰 자동 복구
  - 반품/교환 시 적립금으로 환불
- **쿠폰부정**: 중복사용, 불법양도 적발 시
  - 계정 정지 및 쿠폰 회수

### 8. 쿠폰 조회
- **마이쿠폰**: 사용가능한 쿠폰 목록
- **사용내역**: 사용된 쿠폰 이력
- **통계**: 발급/사용/만료 현황

## 주요 필드
| 필드 | 설명 | 규칙 |
|------|------|------|
| coupon_id | 쿠폰ID | YYMMDDhhmmss+rand4 |
| coupon_cd | 쿠폰코드 | 50자 이내, UNIQUE |
| coupon_nm | 쿠폰명 | 100자 이내 필수 |
| coupon_type_cd | 할인유형 | RATE/FIXED |
| discount_rate | 할인율 | DECIMAL 최대 50% |
| discount_amt | 할인금액 | BIGINT 최대 100,000 |
| min_order_amt | 최소주문금액 | BIGINT 기본값 0 |
| max_discount_amt | 최대할인한도 | BIGINT (정률용) |
| issue_limit | 총발급한도 | INTEGER NULL=무제한 |
| coupon_status_cd | 상태 | COUPON_STATUS 코드 |
| coupon_status_cd_before | 변경전상태 | 상태변경 추적 |
| valid_from | 유효시작 | DATE |
| valid_to | 유효종료 | DATE |
| target_type_cd | 대상 | COUPON_TARGET 코드 |
| target_value | 대상값 | 회원ID/등급코드 |

## 쿠폰 적용 대상 항목 (pm_coupon_item)
- `target_type_cd`: PRODUCT / CATEGORY / VENDOR / BRAND
- 항목이 등록되지 않으면 전체 상품에 적용 (pm_coupon.target_type_cd = ALL)
- 대상이 지정된 경우 해당 상품/카테고리/판매자/브랜드에 속한 상품에만 적용

## 관련 테이블
- `pm_coupon`: 쿠폰 마스터
- `pm_coupon_item`: 쿠폰 적용 대상 (상품/카테고리/판매자/브랜드)
- `pm_coupon_issue`: 회원-쿠폰 발급
- `pm_coupon_usage`: 쿠폰 사용 이력
- `od_order`: 쿠폰 사용내역 (coupon_id)

## 제약사항
- 정률쿠폰의 할인율은 최대 50% 초과 불가
- 중복 사용 방지: 회원당 쿠폰 사용 1회 주문당 1개
- 유효기간 만료 자동 처리
