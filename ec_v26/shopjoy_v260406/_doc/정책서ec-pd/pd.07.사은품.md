# pd.07. 사은품 정책 (상품 관점)

## 정의
주문 조건(금액·수량·등급) 충족 시 **무상**으로 제공하는 품목.
프로모션 도메인(`pm_gift`)에서 지급 조건을 관리하고, 상품 도메인에서 재고와 상품 연결을 관리한다.

> 지급 조건·이력 정책 → `pm.06.프로모션사은품.md` 참조

---

## 사은품 유형

| 유형 | 코드 | 설명 | 상품 연결 |
|---|---|---|---|
| 상품 사은품 | PRODUCT | 실제 판매 상품을 사은품으로 지급 | `pm_gift.prod_id` 연결 |
| 샘플 사은품 | SAMPLE | 별도 판매 없는 샘플 품목 | prod_id 없음 |
| 기타 | ETC | 소모품·패키지 등 | prod_id 없음 |

---

## 재고 관리

- **재고 필드**: `pm_gift.gift_stock`
- **차감 시점**: 주문 결제 완료 후 `pm_gift_issue` 생성 시
- **품절 처리**: `gift_stock = 0` → 사은품 지급 불가, 주문 자체는 계속 진행
- 상품 사은품(`PRODUCT`)의 경우 `pd_prod.prod_stock`도 함께 차감 여부는 운영 정책으로 결정

---

## 주문과의 연결

```
pm_gift_issue (사은품 발급)
  order_id → od_order
  gift_id  → pm_gift

od_order_item (사은품 행)
  gift_id  → pm_gift.gift_id
  unit_price = 0
  prod_nm  = 사은품명 스냅샷
```

- 사은품은 `od_order_item`에 **별도 행**으로 추가 (`unit_price = 0`)
- `od_order_item.gift_id`로 사은품 발급 건과 연결

## 가격·혜택 제한

| 항목 | 정책 |
|---|---|
| 판매가 | 0원 (무상 제공) |
| 쿠폰 적용 | 불가 |
| 할인 적용 | 불가 |
| 적립금 사용 | 불가 |
| 적립 발생 | 불가 |

---

## 배송

- 동일 주문 내 다른 상품과 **합배송 원칙** (별도 배송비 없음)
- 단독 사은품 발송 시 별도 `od_dliv` 생성 + 배송비 부담 주체 결정 필요

---

## 클레임 시 처리

| 클레임 유형 | 사은품 처리 |
|---|---|
| 주문 전체 취소 | 사은품도 자동 취소 (`pm_gift_issue.gift_issue_status_cd = CANCELLED`) + 재고 복원 |
| 일부 취소 (조건 파괴) | 잔여 주문금액 < 지급 조건 → 사은품 회수 또는 시가 청구 |
| 반품 (조건 파괴) | 동일 처리 (회수 또는 시가 청구) |
| 교환 | 사은품은 유지 (교환 대상 상품 변경이므로) |
