# 711. 정산 마감 정책

## 목적
판매자 정산액 계산 및 월별 정산 마감 정책 정의

## 범위
- 정산기간 관리
- 정산대상 선정
- 정산액 계산
- 마감처리

## 정산 주기
- **기본**: 월 1회 (매월 마지막 영업일)
- **정산기간**: 1일 ~ 말일 (한국시간 기준)
- **마감시점**: 매월 마지막 영업일 자정

## 주요 정책

### 1. 정산대상
- **포함**:
  - 배송완료 상태인 주문
  - 반품/교환 완료된 주문
  - 취소 최종 처리된 주문
- **제외**:
  - PENDING, PREPARING 상태 주문
  - 반품 진행 중인 주문
  - 판매자 정지 기간 주문

### 2. 정산액 계산
```
정산액 = 판매액 - 수수료 - 반품환불 - 배송료부담

판매액 = 주문총액 - 할인금 (쿠폰/카테고리 제외)
수수료 = 판매액 × 판매자수수료율
배송료 = 배송사 비용 또는 기본배송료
```

### 3. 정산액 조정
- **배송료정산**: 
  - 고객부담 배송료 회수
  - 판매자부담 배송료 공제
- **반품환불**:
  - 전액반품: 판매액 100% 환불
  - 부분반품: 해당상품금액만 환불
- **할인공제**:
  - 쿠폰할인: 100% 공제 (판매자 부담)
  - 카테고리할인: 50% 공제 (플랫폼/판매자 분담)

### 4. 정산 보류
- **보류사유**:
  - 환불 진행 중 (+7일)
  - 거래 분쟁 (+30일)
  - 판매자 정지 (해제까지)
  - 정산계좌 변경 대기 (확인까지)
- **보류해제**: 사유 제거 후 다음 정산기에 포함

### 5. 정산 세금
- **수익세**: 10% (사업자 등록 여부별)
  - 사업자: 사업자등록증 기준 세금처리
  - 일반인: 10% 원천징수
- **부가세**: 별도 계산 (사업자만 해당)

### 6. 정산 결과 조회
- **정산명세**: PDF 형식 다운로드 가능
  - 기간별 판매액
  - 수수료 계산내역
  - 환불/조정사항
  - 최종정산액
- **정산이력**: 월별 정산내역 조회

### 7. 정산 지연
- **사유**:
  - 계정 미승인 (-1주)
  - 서류 미제출 (-2주)
  - 거래 분쟁 (-1개월)
- **통지**: 사전 공지 (7일 전)

## 정산액 계산 예시
```
주문기간: 2026년 4월
주문금액: 100,000원

1. 판매액 계산
   상품금액: 100,000원
   배송료: 2,500원 (고객부담)
   판매액: 100,000원

2. 수수료 계산 (수수료율 5%)
   수수료: 100,000 × 5% = 5,000원

3. 배송료 처리
   고객배송료: 2,500원 (회수)
   플랫폼배송료: 2,000원 (기본요금)
   배송비순이익: 500원

4. 최종정산액
   정산액 = 100,000 - 5,000 + 500 = 95,500원

5. 세금공제 (사업자가 아닌 경우)
   세금 10%: 95,500 × 10% = 9,550원
   최종지급액: 95,500 - 9,550 = 85,950원
```

## 주요 필드
| 필드 | 설명 | 규칙 |
|------|------|------|
| settle_id | 정산ID | YYMMDDhhmmss+rand4 |
| vendor_id | 판매자ID | FK sy_vendor |
| settle_period | 정산기간 | YYYY-MM |
| settle_amount | 정산액 | BIGINT |
| total_sales | 총판매액 | BIGINT |
| fee_amount | 수수료 | BIGINT |
| refund_amount | 환불액 | BIGINT |
| tax_amount | 세금 | BIGINT |
| final_amount | 최종지급액 | BIGINT |
| settle_status_cd | 상태 | PENDING/COMPLETED/HOLD |
| settle_date | 정산일 | TIMESTAMP |

## 관련 테이블
- sy_vendor: 판매자 정보
- ec_order: 주문 정보
- ec_claim: 반품/환불 정보

## 제약사항
- 정산은 월 1회만 진행
- 정산중 거래 분쟁 발생 시 정산 보류
- 미정산액은 다음달에 이월
