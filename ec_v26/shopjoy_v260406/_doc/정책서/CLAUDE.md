# 정책서/ — 전체 정책서 인덱스

## 폴더 구조

```
정책서/
├─ base/           기반 공통 정책 (UX, 권한, 기술, 인증, 암호화)
├─ ec/
│   ├─ mb/         회원(Member) 도메인
│   ├─ od/         주문·결제·배송·클레임(Order) 도메인
│   ├─ pd/         상품·카테고리·옵션(Product) 도메인
│   ├─ pm/         프로모션(Promotion) 도메인
│   ├─ dp/         전시(Display) 도메인
│   └─ st/         정산(Settlement) 도메인
└─ sy/
    ├─ sy.*.md     시스템 공통·사용자·권한·메뉴·코드·개발표준
    └─ 운영업체/   사이트운영·판매·고객상담·배송·프로그램관리 업체
```
