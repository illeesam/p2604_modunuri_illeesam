# 317. 클레임 - 반품 정책

## 목적
배송 받은 상품의 반품 요청 및 처리 정책 정의

## 범위
- 반품 신청 및 승인
- 반품 수거 및 입고
- 반품 환불 처리
- 반품 배송료 정책

## 반품 상태 (CLAIM_STATUS)
| 상태 | 코드 | 설명 |
|------|------|------|
| 요청 | REQUESTED | 반품 신청 |
| 승인 | APPROVED | 관리자 승인 |
| 수거중 | IN_PICKUP | 수거 진행 중 |
| 입고 | COMPLETED | 입고 완료 |
| 거절 | REJECTED | 반품 불가 |
| 취소 | CANCELLED | 반품 철회 |

## 반품 사유 (CLAIM_REASON)
| 사유 | 코드 | 배송료부담 |
|------|------|-----------|
| 상품하자 | DEFECT | 판매자 100% |
| 배송손상 | DAMAGE | 판매자 100% |
| 오배송 | WRONG_ITEM | 판매자 100% |
| 단순변심 | CHANGE_OF_MIND | 고객 50%, 판매자 50% |
| 사이즈/색상오류 | SIZE_COLOR_ERROR | 판매자 100% |
| 기타 | OTHER | 협의 |

## 주요 정책

### 1. 반품 신청 가능 기간
- **배송완료 후**: 30일 이내
- **자동폐기**: 30일 경과 시 자동거절
- **예외**:
  - 상품하자: 180일 이내 가능
  - 배송손상: 수령 후 7일 이내

### 2. 반품 프로세스
1. **신청**: 고객이 반품 신청
   - 사유 선택 + 상세설명
   - 사진 첨부 (선택, 하자 상품)
2. **접수**: 담당자 검토
   - 기한 내 신청 확인
   - 사유 적정성 검토
3. **승인**: 반품 승인
   - 수거택배사 선정
   - 고객에게 송장번호 전달
4. **수거**: 택배사 픽업
   - 고객 일정 협의
   - 3~5 영업일
5. **입고**: 판매자/회사 입고
   - 상품 검수
   - 문제 있으면 재협의
6. **환불**: 환불금액 결정 후 처리
   - 배송료 공제
   - 손상비용 공제 (필요시)
   - 3~5 영업일 내 환불

### 3. 반품 배송료 정책
- **배송료 부담**:
  - 상품하자/배송손상/오배송: 판매자 100%
  - 단순변심: 고객 50%, 판매자 50%
  - 사이즈/색상오류: 판매자 100%
- **배송료 계산**:
  - 왕복 배송료 기준
  - 도서산간: 실비 + 추가요금
- **배송료 공제**:
  - 환불금액에서 공제
  - 별도 청구 불가

### 4. 반품 검수
- **정상상품**: 태그 미제거, 미사용, 흠집 없음
- **불량상품**: 
  - 작은 흠집: 감액 10~20%
  - 중간 손상: 감액 30~50%
  - 심각 손상: 반품 거절
- **위생상품**: 반품 불가
  - 내의, 수영복, 귀걸이 등

### 5. 반품불가 상품
- **위생상품**: 속옷, 양말 등
- **식품**: 개봉된 상품
- **디지털상품**: 다운로드 완료 상품
- **주문제작**: 완성 후 반품 불가
- **세탁/오염**: 명확한 오염 흔적

### 6. 부분반품
- **수량선택**: 여러 개 중 일부만 반품 가능
- **배송료 재계산**: 
  - 무료배송 기준 미달 시 유료로 전환
  - 추가 배송료 부과 가능

### 7. 환불 처리
- **환불수단**: 원 결제 수단으로
- **환불금액**: 상품가 - 배송료 - 손상비용
- **환불기한**: 입고 후 5~7 영업일
- **부분환불**: 일부 손상 시 감액 환불

## 주요 필드
| 필드 | 설명 | 규칙 |
|------|------|------|
| claim_id | 클레임ID | YYMMDDhhmmss+rand4 |
| claim_type_cd | 클레임유형 | RETURN (고정) |
| claim_status_cd | 반품상태 | CLAIM_STATUS 코드 |
| return_status_cd | 수거상태 | DLIV_STATUS 코드 |
| return_status_cd_before | 변경전상태 | 상태변경 추적 |
| return_shipping_fee | 수거배송료 | BIGINT |
| return_courier_cd | 수거택배사 | COURIER 코드 |
| return_tracking_no | 수거송장 | 택배사 송장번호 |
| inbound_shipping_fee | 입고배송료 | BIGINT |
| inbound_dliv_id | 입고배송ID | ec_dliv FK |
| refund_amt | 환불금액 | BIGINT |
| refund_method_cd | 환불수단 | REFUND_METHOD 코드 |
| reason_cd | 반품사유 | CLAIM_REASON 코드 |
| reason_detail | 상세사유 | TEXT |

## 관련 테이블
- ec_claim: 반품 요청
- ec_claim_item: 반품 상품 명세
- ec_dliv: 수거/입고 배송 정보
- ec_pay: 결제/환불 정보

## 제약사항
- 반품기한(30일) 경과 시 자동거절
- 배송료는 반품 완료 후만 공제
- 상품하자 사진 증거 필수
