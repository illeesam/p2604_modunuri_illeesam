> **목표**: `main` 브랜치에 push 하면 `EcAdminApi` 가 Synology Docker 컨테이너로 자동 배포

## 전체 흐름

```
main push
  → GitHub Actions 트리거
  → [01] 소스코드 체크아웃
  → [02] JDK 17 설정
  → [03] Gradle 빌드 (bootJar)
  → [04] Docker 이미지 빌드
  → [05] 이미지 tar.gz 압축
  → [06] Synology로 SCP 전송
  → [07] 기존 컨테이너 중지/삭제 → 새 이미지 로드 → 컨테이너 실행
  → [08] 결과 요약 출력
  → http://illeesam.synology.me:31000 서비스
```

## Step 3. Synology에 공개키 등록

```bash
# 방법 1: ssh-copy-id 사용 (Git Bash에서)
ssh-copy-id -i ~/.ssh/illeesam260510_ssh.pub -p 22 illeesam@illeesam.synology.me

# 방법 2: 수동 등록
cat ~/.ssh/illeesam260510_ssh.pub
# 출력된 내용을 복사 후 Synology SSH 접속해서 아래 실행
echo "복사한내용" >> ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
```

접속 테스트


## Step 7. Dockerfile 생성

경로: `C:\_pjt_github\p2604_modunuri_illeesam\ec_v26\shopjoy_v260406\_apps_be\EcAdminApi\Dockerfile`

```dockerfile
FROM eclipse-temurin:17-jdk-alpine AS build
WORKDIR /app
COPY . .
RUN ./gradlew bootJar -x test

FROM eclipse-temurin:17-jre-alpine
WORKDIR /app
COPY --from=build /app/build/libs/*.jar app.jar
EXPOSE 31000
ENTRYPOINT ["java", "-jar", "app.jar"]
```

> Maven 프로젝트라면 `./gradlew bootJar -x test` → `./mvnw package -DskipTests` 로 변경

## Step 8. GitHub Actions Workflow 파일 생성

경로: `C:\_pjt_github\p2604_modunuri_illeesam\.github\workflows\deploy-ec-admin.yml`

```yaml
name: Deploy EcAdminApi to Synology

on:
  push:
    branches: [ main ]
    paths:
      - 'ec_v26/shopjoy_v260406/_apps_be/EcAdminApi/**'

jobs:
  deploy:
    runs-on: ubuntu-latest

    steps:

      # ─────────────────────────────────────────
      # [01-01] 소스코드 체크아웃
      # ─────────────────────────────────────────
      - name: "[01-01] 소스코드 체크아웃"
        uses: actions/checkout@v4

      - name: "[01-02] 체크아웃 확인"
        run: |
          echo "======================================"
          echo "[01-02] 체크아웃 완료"
          echo "======================================"
          echo "브랜치  : ${{ github.ref_name }}"
          echo "커밋SHA : ${{ github.sha }}"
          echo "커밋메시지: ${{ github.event.head_commit.message }}"
          echo "작성자  : ${{ github.actor }}"
          echo "시각    : $(date '+%Y-%m-%d %H:%M:%S')"
          echo ""
          echo "[01-02] EcAdminApi 폴더 구조 확인"
          ls -la ec_v26/shopjoy_v260406/_apps_be/EcAdminApi/

      # ─────────────────────────────────────────
      # [02-01] JDK 설정
      # ─────────────────────────────────────────
      - name: "[02-01] JDK 17 설정"
        uses: actions/setup-java@v4
        with:
          java-version: '17'
          distribution: 'temurin'

      - name: "[02-02] Java 버전 확인"
        run: |
          echo "======================================"
          echo "[02-02] Java 환경 확인"
          echo "======================================"
          java -version
          javac -version

      # ─────────────────────────────────────────
      # [03-01] Gradle 빌드
      # ─────────────────────────────────────────
      - name: "[03-01] Gradle 실행 권한 부여"
        working-directory: ec_v26/shopjoy_v260406/_apps_be/EcAdminApi
        run: |
          echo "======================================"
          echo "[03-01] gradlew 실행 권한 부여"
          echo "======================================"
          chmod +x gradlew
          echo "[03-01] 완료"

      - name: "[03-02] Gradle 빌드 (테스트 제외)"
        working-directory: ec_v26/shopjoy_v260406/_apps_be/EcAdminApi
        run: |
          echo "======================================"
          echo "[03-02] Gradle 빌드 시작"
          echo "======================================"
          echo "[03-02] 시작시각: $(date '+%Y-%m-%d %H:%M:%S')"
          ./gradlew bootJar -x test --no-daemon
          echo "[03-02] 종료시각: $(date '+%Y-%m-%d %H:%M:%S')"

      - name: "[03-03] 빌드 결과물 확인"
        working-directory: ec_v26/shopjoy_v260406/_apps_be/EcAdminApi
        run: |
          echo "======================================"
          echo "[03-03] 빌드 결과물 확인"
          echo "======================================"
          ls -lh build/libs/
          JAR_FILE=$(ls build/libs/*.jar | head -1)
          echo "[03-03] 빌드된 JAR: $JAR_FILE"
          echo "[03-03] 파일크기 : $(du -sh $JAR_FILE | cut -f1)"

      # ─────────────────────────────────────────
      # [04-01] Docker 이미지 빌드
      # ─────────────────────────────────────────
      - name: "[04-01] Docker 이미지 빌드"
        working-directory: ec_v26/shopjoy_v260406/_apps_be/EcAdminApi
        run: |
          echo "======================================"
          echo "[04-01] Docker 이미지 빌드 시작"
          echo "======================================"
          echo "[04-01] 시작시각: $(date '+%Y-%m-%d %H:%M:%S')"
          docker build \
            --tag ec-admin-api:latest \
            --tag ec-admin-api:${{ github.sha }} \
            --build-arg BUILD_DATE=$(date '+%Y-%m-%d') \
            --build-arg GIT_COMMIT=${{ github.sha }} \
            .
          echo "[04-01] 종료시각: $(date '+%Y-%m-%d %H:%M:%S')"

      - name: "[04-02] Docker 이미지 확인"
        run: |
          echo "======================================"
          echo "[04-02] 빌드된 Docker 이미지 확인"
          echo "======================================"
          docker images | grep ec-admin-api
          docker inspect ec-admin-api:latest \
            --format '생성시각: {{.Created}} / OS: {{.Os}} / Architecture: {{.Architecture}}'

      # ─────────────────────────────────────────
      # [05-01] Docker 이미지 압축
      # ─────────────────────────────────────────
      - name: "[05-01] Docker 이미지 tar 압축"
        run: |
          echo "======================================"
          echo "[05-01] Docker 이미지 압축 시작"
          echo "======================================"
          echo "[05-01] 시작시각: $(date '+%Y-%m-%d %H:%M:%S')"
          docker save ec-admin-api:latest | gzip > ec-admin-api.tar.gz
          echo "[05-01] 종료시각: $(date '+%Y-%m-%d %H:%M:%S')"
          echo "[05-01] 압축파일 크기: $(du -sh ec-admin-api.tar.gz | cut -f1)"

      # ─────────────────────────────────────────
      # [06-01] Synology로 이미지 전송
      # ─────────────────────────────────────────
      - name: "[06-01] Synology로 Docker 이미지 전송 (SCP)"
        uses: appleboy/scp-action@v0.1.7
        with:
          host: ${{ secrets.SYNOLOGY_HOST }}
          port: ${{ secrets.SYNOLOGY_PORT }}
          username: ${{ secrets.SYNOLOGY_USER }}
          key: ${{ secrets.SYNOLOGY_SSH_KEY }}
          source: "ec-admin-api.tar.gz"
          target: "/volume1/docker/ec-admin-api/"

      - name: "[06-02] 전송 완료 확인"
        uses: appleboy/ssh-action@v1
        with:
          host: ${{ secrets.SYNOLOGY_HOST }}
          port: ${{ secrets.SYNOLOGY_PORT }}
          username: ${{ secrets.SYNOLOGY_USER }}
          key: ${{ secrets.SYNOLOGY_SSH_KEY }}
          script: |
            echo "======================================"
            echo "[06-02] Synology 전송 파일 확인"
            echo "======================================"
            ls -lh /volume1/docker/ec-admin-api/ec-admin-api.tar.gz
            echo "[06-02] 전송 완료 확인 OK"

      # ─────────────────────────────────────────
      # [07-01] Synology 배포
      # ─────────────────────────────────────────
      - name: "[07-01] Synology 컨테이너 배포"
        uses: appleboy/ssh-action@v1
        with:
          host: ${{ secrets.SYNOLOGY_HOST }}
          port: ${{ secrets.SYNOLOGY_PORT }}
          username: ${{ secrets.SYNOLOGY_USER }}
          key: ${{ secrets.SYNOLOGY_SSH_KEY }}
          script: |
            echo "======================================"
            echo "[07-01] 배포 시작"
            echo "[07-01] 시각: $(date '+%Y-%m-%d %H:%M:%S')"
            echo "======================================"

            echo ""
            echo "[07-02] 기존 컨테이너 중지"
            echo "--------------------------------------"
            if docker ps -q --filter name=ec-admin-api | grep -q .; then
              docker stop ec-admin-api
              echo "[07-02] 컨테이너 중지 완료"
            else
              echo "[07-02] 실행 중인 컨테이너 없음 (건너뜀)"
            fi

            echo ""
            echo "[07-03] 기존 컨테이너 삭제"
            echo "--------------------------------------"
            if docker ps -aq --filter name=ec-admin-api | grep -q .; then
              docker rm ec-admin-api
              echo "[07-03] 컨테이너 삭제 완료"
            else
              echo "[07-03] 삭제할 컨테이너 없음 (건너뜀)"
            fi

            echo ""
            echo "[07-04] 기존 이미지 삭제"
            echo "--------------------------------------"
            if docker images -q ec-admin-api:latest | grep -q .; then
              docker rmi ec-admin-api:latest
              echo "[07-04] 이미지 삭제 완료"
            else
              echo "[07-04] 삭제할 이미지 없음 (건너뜀)"
            fi

            echo ""
            echo "[07-05] 새 이미지 로드"
            echo "--------------------------------------"
            cd /volume1/docker/ec-admin-api
            docker load < ec-admin-api.tar.gz
            echo "[07-05] 이미지 로드 완료"
            docker images | grep ec-admin-api

            echo ""
            echo "[07-06] 새 컨테이너 실행"
            echo "--------------------------------------"
            docker run -d \
              --name ec-admin-api \
              --restart unless-stopped \
              --env-file /volume1/docker/ec-admin-api/.env \
              -p 31000:31000 \
              ec-admin-api:latest
            echo "[07-06] 컨테이너 실행 완료"

            echo ""
            echo "[07-07] 컨테이너 상태 확인"
            echo "--------------------------------------"
            sleep 5
            docker ps --filter name=ec-admin-api --format \
              "컨테이너ID: {{.ID}} | 상태: {{.Status}} | 포트: {{.Ports}}"

            echo ""
            echo "[07-08] 컨테이너 초기 로그 확인"
            echo "--------------------------------------"
            docker logs ec-admin-api --tail 30

            echo ""
            echo "[07-09] 임시 파일 정리"
            echo "--------------------------------------"
            rm -f /volume1/docker/ec-admin-api/ec-admin-api.tar.gz
            echo "[07-09] 임시 파일 삭제 완료"

            echo ""
            echo "======================================"
            echo "[07-10] 배포 완료!"
            echo "[07-10] 접속 URL: http://illeesam.synology.me:31000"
            echo "[07-10] 완료시각: $(date '+%Y-%m-%d %H:%M:%S')"
            echo "======================================"

      # ─────────────────────────────────────────
      # [08-01] 배포 결과 요약
      # ─────────────────────────────────────────
      - name: "[08-01] 배포 결과 요약"
        if: always()
        run: |
          echo "======================================"
          echo "[08-01] 배포 파이프라인 결과 요약"
          echo "======================================"
          echo "레포    : ${{ github.repository }}"
          echo "브랜치  : ${{ github.ref_name }}"
          echo "커밋SHA : ${{ github.sha }}"
          echo "작성자  : ${{ github.actor }}"
          echo "결과    : ${{ job.status }}"
          echo "완료시각: $(date '+%Y-%m-%d %H:%M:%S')"
          echo "======================================"
```
