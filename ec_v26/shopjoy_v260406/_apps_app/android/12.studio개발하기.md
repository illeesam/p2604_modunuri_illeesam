# 12. Android Studio 로 개발하기

Android Studio + AVD 에뮬레이터를 사용한 일상 개발 워크플로우.

## Step 1. 프로젝트 import

1. Android Studio 실행 → **Open**
2. `_apps\android\ShopjoyFoApp` 또는 `ShopjoyBoApp` 폴더 선택
3. Gradle Sync 자동 시작 — 의존성 다운로드 (5~10분, 최초만)

## Step 2. AVD (가상 디바이스) 생성

1. **Tools → Device Manager** (또는 우측 사이드바 Device Manager 아이콘)
2. **Create Device** 클릭
3. 권장 디바이스: **Pixel 7** (해상도 1080×2400, API 34)
4. System Image: **Android 14 (API 34) Google Play x86_64**
5. AVD Name: `Pixel_7_API_34`
6. **Finish**

## Step 3. 환경별 Build Variant 선택

좌측 사이드바 **Build Variants** 탭:

| Module | Active Build Variant |
|---|---|
| `app` | `localDebug` / `devDebug` / `prodDebug` / `localRelease` / ... |

`localDebug` 선택 → 호스트 PC `http://10.0.2.2:5501` 가리킴.

## Step 4. 실행

상단 툴바에서:
1. 좌측 드롭다운 → AVD 또는 실기기 선택
2. ▶ 또는 **Run → Run 'app'** (Shift+F10)
3. AVD 부팅 → 자동 설치 → 자동 실행

첫 실행 시 AVD 부팅에 1~2분 소요.

## Step 5. WebView 디버깅 (Chrome DevTools)

1. 에뮬레이터/실기기에서 ShopJoy 앱 실행 중
2. 데스크톱 Chrome → `chrome://inspect/#devices`
3. ShopJoy 앱 항목 → **inspect** 클릭
4. 일반 Chrome DevTools 와 동일하게 사용

> Vue DevTools 설치하면 Vue 3 컴포넌트 트리도 검사 가능 (Chrome 확장).

## Step 6. 라이브 리로드 워크플로우

1. **VS Code Live Server** 실행 (포트 5501) — 별도 창
2. Android Studio 에서 `localDebug` 변형 선택 + ▶ 실행
3. 웹 코드 수정 (`pages/Home01.js` 등) → 저장
4. 에뮬레이터/핸드폰 ShopJoy 앱 화면 새로고침 (당겨서 새로고침 또는 메뉴)

> 주의: 네이티브 코드 (Kotlin / Manifest / build.gradle.kts) 변경 시 ▶ 다시 빌드 필요.

## Step 7. 일반 Kotlin 디버깅

`MainActivity.kt` 등에 브레이크포인트 → ▶ 옆 **Debug** 버튼 (Shift+F9) → 디버거 연결.

| 단축키 | 동작 |
|---|---|
| F8 | Step Over |
| F7 | Step Into |
| F9 | Resume |
| Ctrl+F8 | 브레이크포인트 토글 |
| Ctrl+F2 | Stop |

## Step 8. 자주 쓰는 Gradle 작업

| Task | 설명 |
|---|---|
| `assembleLocalDebug` | local debug APK 빌드 |
| `assembleProdRelease` | prod release APK 빌드 |
| `bundleProdRelease` | prod release **AAB** 빌드 (Play Store 용) |
| `installLocalDebug` | local debug APK 빌드 + 연결된 기기 설치 |
| `clean` | 빌드 산출물 정리 |
| `dependencies` | 의존성 트리 출력 |

실행 (Android Studio):
- **View → Tool Windows → Gradle** 패널 → 모듈 트리에서 더블클릭
- 또는 터미널: `.\gradlew assembleLocalDebug`

## 트러블슈팅

| 증상 | 해결 |
|---|---|
| Gradle Sync 무한 대기 | **File → Invalidate Caches / Restart** |
| AVD 시작 실패 (HAXM 오류) | BIOS 에서 Intel VT-x / AMD-V 활성화 |
| AVD 매우 느림 | x86_64 이미지 사용 (ARM 이미지 X), 8GB+ RAM 할당 |
| `Hot Swap not supported` | Apply Changes 대신 Run 다시 |
| WebView 가 흰 화면 | Live Server 켜져 있는지, 포트 5501 방화벽 허용 |
