# 22. (참고) iOS 설정하기

`_apps/android/` 는 안드로이드 전용이라 iOS 설정은 [`_apps/ios/22.IOS설정하기.md`](../ios/22.IOS설정하기.md) 를 참조하세요.

## 안드로이드 앱과 iOS 앱의 관계

ShopJoy 모바일은 **안드로이드와 iOS가 별도 앱**입니다 (다른 폴더 / 다른 기술 / 다른 빌드).

| 구분 | 위치 | 기술 |
|---|---|---|
| 순수 네이티브 안드로이드 | `_apps/android/` | Kotlin |
| 순수 네이티브 iOS | `_apps/ios/` | Swift |
| 크로스플랫폼 (둘 다 한 번에) | `_apps/capacitor/`, `_apps/reactNative/`, `_apps/flutter/` | JavaScript / TypeScript / Dart |

만약 **iOS 와 동시 출시** 가 필요하다면:

1. 같은 BO/FO 기능을 양쪽에서 구현 → 유지보수 비용 2배
2. 또는 **`_apps/capacitor/` / `_apps/reactNative/` / `_apps/flutter/`** 로 단일 코드베이스 운영 권장

## iOS 만 별도 정책 (참고)

| 항목 | 안드로이드 | iOS |
|---|---|---|
| 개발 머신 | Windows / macOS / Linux | **macOS 필수** |
| 가입비 | Google Play USD $25 (1회) | Apple Developer USD $99/년 |
| 인증서 관리 | 키스토어 (`.keystore` 또는 `.jks`) | `.p12` + Provisioning Profile |
| 검토 시간 | 1~3일 | 1~3일 (드물게 1주) |
| 푸시 시스템 | FCM | APNs (FCM도 가능) |
