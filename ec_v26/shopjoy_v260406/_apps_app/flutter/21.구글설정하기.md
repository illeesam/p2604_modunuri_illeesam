ShopJoy Flutter 앱에 FCM 푸시 추가.

## 1단계: FlutterFire CLI 설치

```bash
dart pub global activate flutterfire_cli
```

확인:
```bash
flutterfire --version
```

## 2단계: Firebase CLI 로그인

```bash
npm install -g firebase-tools
firebase login
```

브라우저 → Google 로그인.

---

## 3단계: Firebase 프로젝트 생성 + 등록

### 3-1. 프로젝트 생성

https://console.firebase.google.com → 프로젝트 추가 → `shopjoy-prod`.

### 3-2. flutterfire configure 실행

```bash
cd _apps/flutter/ShopjoyFoApp
flutterfire configure
```

대화형:
1. 프로젝트 선택 (`shopjoy-prod`)
2. 플랫폼 선택 (`android`, `ios`)
3. Android applicationId 자동 인식 (`com.shopjoy.fo`)
4. iOS Bundle ID 자동 인식 (`com.shopjoy.fo`)

자동 생성:
- `android/app/google-services.json`
- `ios/Runner/GoogleService-Info.plist`
- `lib/firebase_options.dart`


## 5단계: main.dart 에 Firebase 초기화

`lib/main.dart`:

```dart
import 'package:firebase_core/firebase_core.dart';
import 'firebase_options.dart';

Future<void> main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await Firebase.initializeApp(
    options: DefaultFirebaseOptions.currentPlatform,
  );
  await dotenv.load(fileName: AppEnv.envFile);
  runApp(const ShopjoyFoApp());
}
```

## 8단계: 푸시 코드 작성

`lib/push.dart` 생성:

```dart
import 'dart:io';
import 'package:firebase_messaging/firebase_messaging.dart';
import 'package:flutter/foundation.dart';

class PushManager {
  static final _messaging = FirebaseMessaging.instance;

  static Future<void> initialize() async {
    // 권한 요청
    final settings = await _messaging.requestPermission(
      alert: true,
      badge: true,
      sound: true,
    );
    if (settings.authorizationStatus != AuthorizationStatus.authorized &&
        settings.authorizationStatus != AuthorizationStatus.provisional) {
      return;
    }

    // 토큰 발급
    final token = await _messaging.getToken();
    debugPrint('[FCM] token: $token');
    // TODO: 백엔드 등록 — POST /fo/my/device-token

    // 토큰 갱신 시
    _messaging.onTokenRefresh.listen((newToken) {
      debugPrint('[FCM] refreshed: $newToken');
    });

    // 포그라운드 메시지
    FirebaseMessaging.onMessage.listen((message) {
      debugPrint('[FCM] foreground: ${message.notification?.title}');
    });

    // 푸시 탭 (백그라운드 → 포그라운드)
    FirebaseMessaging.onMessageOpenedApp.listen((message) {
      debugPrint('[FCM] tapped: ${message.data}');
      // TODO: WebView 에 onAppDeepLink JS 호출
    });
  }
}
```

`lib/main.dart` 의 `runApp` 직전에 추가:

```dart
await PushManager.initialize();
```

### 10-1. 토큰 출력 확인

```bash
flutter run --dart-define=APP_ENV=local
```

Cloud Messaging → 새 캠페인 → 알림 → 테스트 메시지 보내기 → 토큰 입력 → 테스트.

핸드폰에 푸시 도착 확인.

## 트러블슈팅

| 증상 | 해결 |
|---|---|
| `firebase_options.dart` 없음 | `flutterfire configure` 다시 |
| Android `Default FirebaseApp not initialized` | `main()` 에서 `Firebase.initializeApp` 호출 누락 |
| iOS 토큰 nil | 시뮬레이터 X (실기기 필수), APNs 키 업로드 확인 |
| Push 도착 X | Firebase 프로젝트 / 토큰 일치 확인 |
| `Build error: 'Firebase/Core.h' not found` | `cd ios && pod deintegrate && pod install` |
