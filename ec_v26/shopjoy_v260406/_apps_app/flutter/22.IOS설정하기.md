## 핵심 단계 요약

### 1. App Identifier 등록

https://developer.apple.com → Identifiers → **+** → 환경별 Bundle ID:
- `com.shopjoy.fo`
- (분리하려면) `com.shopjoy.fo.local` / `com.shopjoy.fo.dev`
- BO 도 동일

Capabilities → **Push Notifications** ✅.

### 2. APNs 인증 키 발급

Keys → **+** → APNs → `.p8` 다운로드 → Key ID + Team ID 메모.

### 3. Distribution Certificate

Certificates → **+** → Apple Distribution → CSR 업로드 → 다운로드 → 키체인 등록.

### 4. Provisioning Profile

Profiles → **+** → App Store → Bundle ID + Certificate → 다운로드 → 더블클릭 (Xcode 자동 등록).

### 5. Xcode 자동 서명 설정

```bash
cd _apps/flutter/ShopjoyFoApp
open ios/Runner.xcworkspace
```

Xcode → Runner 타겟 → **Signing & Capabilities**:
- ✅ Automatically manage signing
- Team 선택

### 6. Bundle ID 확인 / 변경

Xcode → Runner → General → Identity → Bundle Identifier:
- 기본: `com.example.shopjoyFoApp`
- 변경: `com.shopjoy.fo`

또는 `ios/Runner.xcodeproj/project.pbxproj` 직접 편집.

### 7. Push Notifications + Background Modes Capability

**+ Capability**:
- Push Notifications
- Background Modes → ✅ Remote notifications

### 8. APNs 키 Firebase 업로드

[`21.구글설정하기.md`](21.구글설정하기.md) 의 **6단계** 참조.

### 9. App Store Connect 앱 생성

https://appstoreconnect.apple.com → 나의 앱 → **+** → 새 앱.

## 환경별 분리 (Flavor)

Flutter 는 **Flavor** 로 환경 분리. `android/app/build.gradle` + iOS Configuration:

### Android (build.gradle)

```gradle
android {
    flavorDimensions "default"
    productFlavors {
        local { dimension "default"; applicationIdSuffix ".local" }
        dev   { dimension "default"; applicationIdSuffix ".dev" }
        prod  { dimension "default" }
    }
}
```

빌드:
```bash
flutter run --flavor local --dart-define=APP_ENV=local
flutter run --flavor prod --dart-define=APP_ENV=prod
```

### iOS Scheme

Xcode → Manage Schemes → **+** Duplicate:
- `ShopjoyFoApp-Local`
- `ShopjoyFoApp-Dev`
- `ShopjoyFoApp-Prod`

각 Scheme 의 Build Configuration 별로 PRODUCT_BUNDLE_IDENTIFIER 설정.

빌드:
```bash
flutter run --flavor local
```
