# 11. Flutter 개발환경 구축 — 따라하기

## 사전 준비

- [ ] Windows 10/11 / macOS 14+ / Ubuntu 22+
- [ ] 디스크 50 GB 이상

---

## 1단계: Flutter SDK 설치

### Windows

```powershell
# Chocolatey 사용
choco install flutter

# 또는 수동:
# 1. https://docs.flutter.dev/get-started/install/windows 에서 zip 다운로드
# 2. C:\dev\flutter 에 압축 해제
# 3. PATH 에 C:\dev\flutter\bin 추가
```

### macOS

```bash
brew install --cask flutter
```

### 확인

```bash
flutter --version
```

다음과 같이 보이면 OK:

```
Flutter 3.22.x
Dart 3.4.x
```

---

## 2단계: Android 환경

### 2-1. JDK 17

**Windows**: `winget install EclipseAdoptium.Temurin.17.JDK`
**macOS**: `brew install --cask temurin@17`

### 2-2. Android Studio

https://developer.android.com/studio 설치.

### 2-3. Android SDK 환경변수

위 [Capacitor 11.개발환경.md](../capacitor/11.개발환경.md) 의 3단계 그대로.

### 2-4. Flutter 가 Android 인식 확인

```bash
flutter doctor --android-licenses
# 모든 라이선스 y 동의

flutter doctor
```

다음 항목 모두 ✓ :

```
[✓] Flutter
[✓] Android toolchain
[✓] Chrome (선택)
[✓] Android Studio
```

---

## 3단계: iOS 환경 (macOS 만)

### 3-1. Xcode + CLT

App Store → Xcode → `xcode-select --install` → `sudo gem install cocoapods`.

### 3-2. Flutter 가 iOS 인식 확인

```bash
flutter doctor
```

`[✓] Xcode` 표시 확인.

---

## 4단계: VS Code Flutter 확장 설치 (권장)

VS Code → 확장 → `Flutter` (Dart-Code) 검색 → Install.

자동으로 Dart 확장도 함께 설치됨.

---

## 5단계: ShopJoy Flutter 프로젝트 의존성 설치

### 5-1. 네이티브 디렉토리 생성

```bash
cd _apps/flutter/ShopjoyFoApp
flutter create .
```

`android/` `ios/` `linux/` `macos/` `windows/` 자동 생성.

> 이미 존재하면 skip.

### 5-2. pubspec 의존성 설치

```bash
flutter pub get
```

성공 출력:

```
Resolving dependencies...
Got dependencies!
```

### 5-3. BO 도

```bash
cd ../ShopjoyBoApp
flutter create .
flutter pub get
```

---

## 6단계: 환경 분기 동작 확인

```bash
cd _apps/flutter/ShopjoyFoApp
flutter run --dart-define=APP_ENV=local
```

에뮬레이터/시뮬레이터 부팅 + ShopJoy 앱 실행되면 환경 구축 완료!

---

## 검증 체크리스트

- [ ] `flutter --version` 3.22+
- [ ] `flutter doctor` 모든 항목 통과
- [ ] `flutter pub get` 성공
- [ ] `flutter run` 으로 앱 실행 성공
