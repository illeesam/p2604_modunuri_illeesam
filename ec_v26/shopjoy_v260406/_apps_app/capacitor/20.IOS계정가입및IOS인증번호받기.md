# 20. Apple ID 가입 + iOS 인증번호 받기 — 따라하기 (Capacitor)

> Capacitor iOS 빌드는 결국 Xcode 프로젝트 (`ios/App/App.xcworkspace`) 라서 절차가 순수 iOS 와 동일합니다.

이 문서는 **Capacitor 관점에서 핵심 단계** 만 정리합니다. 자세한 절차는 → [`_apps/ios/20.IOS계정가입및IOS인증번호받기.md`](../ios/20.IOS계정가입및IOS인증번호받기.md)

## iOS 인증번호 — 한눈에

| 용도 | iOS 식별자 |
|---|---|
| 앱 식별 | **Bundle ID** = Capacitor `appId` |
| 팀 식별 | **Team ID** (10자) |
| 푸시 발송 | **APNs 인증 키 (.p8)** + Key ID |
| 코드 서명 | **Distribution Certificate** |
| 빌드 권한 | **Provisioning Profile** |

## 사전 준비

- [ ] **macOS 14+** + Xcode 15.4+
- [ ] 신용카드 (Apple Developer Program $99/년)

---

## 1단계: Apple ID 가입

브라우저에서 https://appleid.apple.com → **Apple ID 생성** → 이메일/비밀번호/전화 인증 → 가입 완료.

**2단계 인증 필수** — https://appleid.apple.com → **로그인 및 보안 → 2단계 인증 → 켜기**.

운영용 / 개인용 Apple ID 분리 권장.

---

## 2단계: Apple Developer Program 가입

https://developer.apple.com/programs/enroll → 시작하기 → Apple ID 로그인 → **Individual** 또는 **Organization** → 결제 ($99) → 1~2 영업일 승인 대기.

---

## 3단계: Team ID 확인

1. https://developer.apple.com/account → 좌측 **Membership**
2. **Team ID** (10자, 예: `ABCDE12345`) 메모

---

## 4단계: Bundle ID (App Identifier) 등록

Capacitor 의 `capacitor.config.ts` 의 `appId` 와 일치해야 함:

| 환경 | Bundle ID | Capacitor `appId` |
|---|---|---|
| FO | `com.shopjoy.fo` | `appId: 'com.shopjoy.fo'` |
| BO | `com.shopjoy.bo` | `appId: 'com.shopjoy.bo'` |

### 등록 절차

1. https://developer.apple.com/account → **Identifiers** → **+**
2. **App IDs** → **App** → **Continue**
3. Bundle ID: `com.shopjoy.fo` (Explicit)
4. Capabilities ✅ **Push Notifications** + **Sign in with Apple** (사용 시)
5. **Continue → Register**

> Capacitor 는 환경별 분리 안 하고 `appId` 단일 사용이 기본. 분리하려면 `capacitor.config.ts` 의 `appId` 를 환경별로 수정 + 각 Bundle ID 별도 등록.

---

## 5단계: APNs 인증 키 (.p8) 발급

Capacitor 의 `@capacitor/push-notifications` 가 사용.

1. https://developer.apple.com/account → **Keys** → **+**
2. Key Name: `ShopJoy APNs Key`
3. ✅ **Apple Push Notifications service (APNs)** → **Configure** → **All Apps** 선택 → **Save**
4. **Continue → Register → Download** → `AuthKey_XXXX.p8` 다운로드 (한 번만)
5. **Key ID** + **Team ID** 메모

---

## 6단계: Distribution Certificate

Mac 키체인 접근 → **인증서 지원 → 인증 기관에서 인증서 요청** → CSR 파일 생성 → Apple Developer Portal **Certificates → +** → **Apple Distribution** → CSR 업로드 → 다운로드 → 더블클릭 → 키체인 등록.

자세한 절차는 [`_apps/ios/20.IOS계정가입및IOS인증번호받기.md`](../ios/20.IOS계정가입및IOS인증번호받기.md) 의 6단계 참조.

---

## 7단계: Provisioning Profile

Apple Developer Portal **Profiles → +** → **App Store** → Bundle ID + Distribution Certificate 선택 → 다운로드 → 더블클릭 → Xcode 자동 등록.

---

## 8단계: Xcode 자동 서명 (Capacitor 의 `ios/App`)

```bash
cd _apps/capacitor/ShopjoyFoApp
npm run sync:prod              # capacitor.config.ts → server.url 운영 + cap sync ios
npm run open:ios               # Xcode 열기 (App.xcworkspace)
```

Xcode 좌측 → **App** 타겟 → **Signing & Capabilities**:
- ✅ Automatically manage signing
- Team 선택
- Bundle Identifier: `com.shopjoy.fo` (Capacitor `appId` 와 일치 확인)

**+ Capability**:
- ✅ Push Notifications
- ✅ Background Modes → Remote notifications

---

## 9단계: Firebase 에 APNs 키 업로드

Capacitor 푸시는 결국 FCM 으로 통합됨.

1. https://console.firebase.google.com → 프로젝트 → ⚙ → 프로젝트 설정
2. **클라우드 메시징** 탭 → **Apple 앱 구성** → **APNs 인증 키 업로드**
3. `.p8` 파일 + Key ID + Team ID 입력 → **업로드**

상세 → [`21.구글설정하기.md`](21.구글설정하기.md) **4-4단계**.

---

## 10단계: GoogleService-Info.plist 배치

Firebase Console → iOS 앱 (`com.shopjoy.fo`) 추가 → `GoogleService-Info.plist` 다운로드:

```bash
mv ~/Downloads/GoogleService-Info.plist \
   _apps/capacitor/ShopjoyFoApp/ios/App/App/GoogleService-Info.plist
```

Xcode → 우클릭 → **Add Files to "App"** → 선택 → **Copy items if needed** ✅ → **Add to targets: App** ✅.

---

## 11단계: App Store Connect 앱 생성

https://appstoreconnect.apple.com → 나의 앱 → **+** → 새 앱:
- 플랫폼: iOS
- 이름: ShopJoy
- Bundle ID: `com.shopjoy.fo`
- SKU: `SHOPJOY-FO-001`

---

## 12단계: 카카오 SDK iOS (사용 시)

https://developers.kakao.com → 내 애플리케이션 → **플랫폼 → iOS 플랫폼 등록**:
- **Bundle ID**: `com.shopjoy.fo` (각 환경별)
- iOS 는 키 해시 등록 없음

---

## 검증

```bash
cd _apps/capacitor/ShopjoyFoApp
npm run ios:dev               # 또는 ios:prod
```

Xcode 에서 자동 서명 OK + 시뮬레이터 / 실기기 실행 확인.

iPhone 실기기 (시뮬레이터 X) → APNs 토큰 콘솔 출력 → Firebase 테스트 푸시 도착.
