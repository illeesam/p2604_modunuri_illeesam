# 21. Google (Firebase) 설정하기 — 따라하기

ShopJoy iOS 앱에 FCM 푸시 알림을 통합하는 절차.

> iOS 푸시는 Apple APNs 가 기본이지만, 안드로이드와 통일성을 위해 **Firebase Cloud Messaging (FCM)** 을 통해 발송합니다.

- [ ] Apple Developer 유료 계정 ($99/년)

## 1단계: Firebase 콘솔 프로젝트 생성

### 1-1. 콘솔 접속

브라우저에서 https://console.firebase.google.com 접속 → Google 로그인.

### 1-2. 프로젝트 추가

**프로젝트 추가** 버튼 → 프로젝트 이름 입력 (예: `shopjoy-prod`) → **계속**.

### 1-3. Google Analytics 설정

OFF 선택 (필요 시 나중에 추가) → **프로젝트 만들기** → 1~2분 대기 → **계속**.

## 2단계: iOS 앱 등록

### 2-1. iOS 추가

프로젝트 대시보드 → 좌측 상단 ⚙ **프로젝트 설정** → **앱 추가** → **iOS** 아이콘 클릭.

### 2-2. Apple Bundle ID 입력

| 환경 | Bundle ID |
|---|---|
| FO Local | `com.shopjoy.fo.local` |
| FO Dev | `com.shopjoy.fo.dev` |
| FO Prod | `com.shopjoy.fo` |
| BO Local | `com.shopjoy.bo.local` |
| BO Dev | `com.shopjoy.bo.dev` |
| BO Prod | `com.shopjoy.bo` |

각 환경별로 별도 등록 (FO 만 사용하면 3개, BO 까지 6개).

**앱 닉네임**: `ShopJoy FO Prod` 등 (선택사항).

**App Store ID**: 빈칸 (App Store 등록 후 입력).

→ **앱 등록**.

### 2-3. GoogleService-Info.plist 다운로드

다음 페이지에서 **GoogleService-Info.plist 다운로드** 버튼 클릭.

## 4단계: APNs 인증 키 업로드

### 4-1. APNs 키 발급 (Apple Developer)

먼저 [`22.IOS설정하기.md`](22.IOS설정하기.md) 의 **2단계: APNs 인증 키 생성** 을 완료해야 합니다.

발급물:
- `.p8` 파일 (예: `AuthKey_ABCD123456.p8`)
- Key ID (예: `ABCD123456`)
- Team ID (예: `1A2B3C4D5E`)

### 4-2. Firebase Console 에 업로드

Firebase Console → 프로젝트 설정 → **클라우드 메시징** 탭 → **Apple 앱 구성** 섹션.

**APNs 인증 키 업로드**:

1. **업로드** 클릭
2. `.p8` 파일 선택
3. **Key ID** 입력
4. **Team ID** 입력
5. **업로드**

## 5단계: Firebase SDK 설치

### 5-1. Podfile 생성

```bash
cd _apps/ios/ShopjoyFoApp/ShopjoyFoApp
pod init
```

`Podfile` 이 생성됨. 다음과 같이 편집:

```ruby
platform :ios, '14.0'

target 'ShopjoyFoApp' do
  use_frameworks!

  pod 'FirebaseCore'
  pod 'FirebaseMessaging'
end
```

### 5-2. Pod 설치

```bash
pod install
```

다음과 같이 보이면 OK:

```
Installing FirebaseCore (11.x.x)
Installing FirebaseMessaging (11.x.x)
...
Pod installation complete!
```

### 5-3. Xcode workspace 사용

이후로는 `.xcodeproj` 가 아닌 `.xcworkspace` 를 열어야 함:

```bash
open ShopjoyFoApp.xcworkspace
```

## 6단계: AppDelegate 코드 활성화

`ShopjoyFoApp/AppDelegate.swift` 편집:

```swift
import UIKit
import FirebaseCore       // ← 주석 해제
import FirebaseMessaging  // ← 주석 해제

@main
class AppDelegate: UIResponder, UIApplicationDelegate {

    func application(_ application: UIApplication,
                     didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        FirebaseApp.configure()           // ← 주석 해제
        Messaging.messaging().delegate = self
        registerForPushNotifications(application)
        return true
    }

    func application(_ application: UIApplication,
                     didRegisterForRemoteNotificationsWithDeviceToken deviceToken: Data) {
        Messaging.messaging().apnsToken = deviceToken    // ← 주석 해제
    }

    private func registerForPushNotifications(_ application: UIApplication) {
        UNUserNotificationCenter.current().requestAuthorization(options: [.alert, .badge, .sound]) { granted, _ in
            guard granted else { return }
            DispatchQueue.main.async { application.registerForRemoteNotifications() }
        }
    }
}

extension AppDelegate: MessagingDelegate {
    func messaging(_ messaging: Messaging, didReceiveRegistrationToken fcmToken: String?) {
        guard let token = fcmToken else { return }
        print("[FCM] token: \(token)")
        // TODO: 백엔드에 토큰 등록 (POST /fo/my/device-token)
    }
}
```

## 9단계: 테스트 메시지 발송

### 9-1. Firebase Console → Cloud Messaging

좌측 메뉴 → **Cloud Messaging** → **새 캠페인** → **알림**.

### 9-2. 알림 작성

- 알림 제목: `ShopJoy 테스트`
- 알림 텍스트: `푸시 동작 검증`

→ **다음** → **다음** → **다음**...

### 9-3. 테스트 메시지 보내기

마지막 단계 직전 → **테스트 메시지 보내기** 버튼 → FCM 토큰 입력 → **테스트**.

iPhone 화면에 푸시 알림 표시되면 성공!

## 10단계: 토큰 백엔드 등록 (선택)

`AppDelegate.swift` 의 `messaging(_:didReceiveRegistrationToken:)` 안에서:

```swift
guard let url = URL(string: "https://api.shopjoy.com/api/fo/my/device-token") else { return }
var request = URLRequest(url: url)
request.httpMethod = "POST"
request.setValue("application/json", forHTTPHeaderField: "Content-Type")
let body: [String: Any] = [
    "pushToken": token,
    "deviceType": "ios",
    "deviceModel": UIDevice.current.model,
    "deviceOs": "iOS \(UIDevice.current.systemVersion)",
    "appVersion": Bundle.main.object(forInfoDictionaryKey: "CFBundleShortVersionString") as? String ?? "1.0.0"
]
request.httpBody = try? JSONSerialization.data(withJSONObject: body)
URLSession.shared.dataTask(with: request) { _, _, error in
    if let e = error { print("[token reg] failed: \(e)") }
    else { print("[token reg] OK") }
}.resume()
```

## 트러블슈팅

| 증상 | 해결 |
|---|---|
| `FirebaseApp not configured` | `FirebaseApp.configure()` 호출 누락 |
| 토큰이 nil | APNs 인증 키 미등록 / 실기기 아닌 시뮬레이터 |
| 푸시 권한 다이얼로그 안 뜸 | Info.plist 의 `NSUserNotificationsUsageDescription` 확인 |
| 푸시 발송했는데 도착 안 함 | iPhone 푸시 권한 허용? 백그라운드 새로고침 ON? |
| `pod install` 실패 (M1/M2) | `arch -x86_64 pod install` |
