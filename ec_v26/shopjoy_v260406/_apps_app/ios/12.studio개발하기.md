# 12. Xcode 로 개발하기 — 따라하기

Xcode 시뮬레이터로 ShopJoy iOS 앱을 일상 개발하는 워크플로우.

## 사전 준비

- [ ] [`11.개발환경.md`](11.개발환경.md) 완료
- [ ] Mac + Xcode + XcodeGen 설치 완료
- [ ] VS Code + Live Server 확장 설치 완료

---

## 1단계: 프로젝트 열기

### 1-1. 터미널에서 프로젝트 디렉토리로 이동

```bash
cd _apps/ios/ShopjoyFoApp
```

### 1-2. Xcode 프로젝트 자동 생성 + 열기

```bash
xcodegen generate
open ShopjoyFoApp.xcodeproj
```

Xcode 창이 뜨면 다음 단계로.

---

## 2단계: 시뮬레이터 선택

### 2-1. Xcode 창 좌상단 클릭

Scheme 드롭다운(`ShopjoyFoApp-Local`) 옆 디바이스 드롭다운 클릭.

### 2-2. iPhone 15 Pro 선택

목록에서 **iPhone 15 Pro** 클릭.

목록에 없으면:
- Xcode → **Window → Devices and Simulators** → **Simulators** 탭 → **+** → 새 시뮬레이터 추가

---

## 3단계: 환경 선택 (local / dev / prod)

좌상단 Scheme 드롭다운 클릭 → 원하는 환경 선택:

- `ShopjoyFoApp-Local` — 호스트 PC `http://127.0.0.1:5501` 가리킴
- `ShopjoyFoApp-Dev` — Netlify 개발
- `ShopjoyFoApp-Prod` — Netlify 운영

`local` 선택 시 → 다음 단계.

---

## 4단계: VS Code Live Server 시작

별도 터미널에서:

```bash
cd /path/to/shopjoy_v260406
code .
```

VS Code 가 열리면:
1. `index.html` 우클릭 → **Open with Live Server**
2. 브라우저가 자동으로 열림 → 포트 5501 동작 중 확인

---

## 5단계: ▶ 빌드 + 실행

Xcode 좌상단 ▶ 버튼 클릭 (단축키 **Cmd + R**).

진행 상황:
1. Build 진행 (5~30초)
2. 시뮬레이터 자동 부팅 (첫 실행 시 1~2분)
3. ShopJoy 앱 자동 설치
4. ShopJoy 앱 자동 실행

ShopJoy 화면이 시뮬레이터에 보이면 성공!

---

## 6단계: 코드 수정 → 자동 반영 (라이브 리로드)

웹 코드 (`pages/Home01.js`, `assets/css/...`) 를 수정하고 저장하면:

- VS Code Live Server 가 변경 감지 → 자동으로 페이지 새로고침
- iOS 시뮬레이터의 ShopJoy 도 자동 새로고침

수동 새로고침이 필요하면 시뮬레이터에서 **Cmd+R** (Web Inspector 가 열려있을 때).

---

## 7단계: WebView 디버깅 (Safari Web Inspector)

### 7-1. Mac Safari 개발자 메뉴 활성화

Safari → **Safari → 환경설정 → 고급** → **메뉴 막대에 개발자용 메뉴 표시** 체크.

### 7-2. iOS 시뮬레이터에서 ShopJoy 앱 실행 중일 때

Mac Safari → 메뉴바 → **개발자용 → Simulator → iPhone 15 Pro → ShopJoy.app**

### 7-3. Web Inspector 사용

데스크톱 브라우저 DevTools 와 똑같이 사용:
- **Console** 탭: `console.log` 결과 확인
- **Network** 탭: API 호출 확인
- **Sources** 탭: 브레이크포인트 + 디버깅
- **Elements** 탭: DOM 검사

---

## 8단계: 네이티브 디버깅 (Xcode 콘솔)

Xcode 메뉴 → **View → Debug Area → Show Debug Area** (Cmd+Shift+Y)

하단 Debug Area 가 열리면:
- `print(...)` 출력 확인
- 네이티브 에러 / 크래시 메시지 표시

---

## 9단계: 자주 쓰는 단축키

| 단축키 | 동작 |
|---|---|
| **Cmd + R** | 빌드 + 실행 |
| **Cmd + B** | 빌드만 |
| **Cmd + .** | 실행 중지 |
| **Cmd + Shift + K** | Clean Build Folder |
| **Cmd + Shift + Y** | Debug Area 토글 |
| **Cmd + 0** | Project Navigator 토글 |
| **Cmd + 1~9** | 좌측 사이드바 탭 전환 |

---

## 10단계: 코드 자동완성 / Search

| 단축키 | 동작 |
|---|---|
| **Cmd + Shift + O** | 파일 빠르게 열기 |
| **Cmd + Shift + F** | 전체 프로젝트 텍스트 검색 |
| **Cmd + Click** | 정의로 이동 |
| **Ctrl + Cmd + ←/→** | 이전/다음 위치 이동 |

---

## 트러블슈팅

| 증상 | 해결 |
|---|---|
| Build 무한 대기 | Xcode → **Product → Clean Build Folder** (Cmd+Shift+K) → 다시 ▶ |
| 시뮬레이터 흰 화면 | local 환경인데 Live Server 가 꺼짐 → VS Code 에서 다시 시작 |
| 시뮬레이터 매우 느림 | `xcrun simctl shutdown all` → 시뮬레이터 다시 부팅 |
| `Could not load module` | Xcode → **File → Packages → Reset Package Caches** |
| `xcodegen` 실행 후 변경 미반영 | Xcode 닫고 → `xcodegen generate` → 다시 `open` |
| 코드 자동완성이 안 됨 | **File → Packages → Resolve Package Versions** 또는 Xcode 재시작 |
