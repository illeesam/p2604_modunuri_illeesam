# 22. Apple Developer 설정 (RN) — 따라하기

ShopJoy RN iOS 앱의 Apple Developer 인증서 / 프로파일.

> RN iOS 는 결국 Xcode 프로젝트 (`ios/ShopjoyFoApp.xcworkspace`) 라서 절차 동일.
> 자세한 절차 → [`_apps/ios/22.IOS설정하기.md`](../ios/22.IOS설정하기.md)

## 사전 준비

- [ ] [`11.개발환경.md`](11.개발환경.md) 완료
- [ ] macOS + Xcode
- [ ] Apple Developer Program ($99/년)

---

## 핵심 단계 요약

### 1. App Identifier 등록

https://developer.apple.com → Identifiers → **+** → 각 환경별 Bundle ID:
- `com.shopjoy.fo.local` / `com.shopjoy.fo.dev` / `com.shopjoy.fo`
- BO 도 동일

Capabilities → **Push Notifications** ✅.

### 2. APNs 인증 키 발급

Keys → **+** → APNs → `.p8` 다운로드 → Key ID + Team ID 메모.

### 3. Distribution Certificate

Certificates → **+** → Apple Distribution → CSR 업로드 → 다운로드 → 키체인 등록.

### 4. Provisioning Profile

Profiles → **+** → App Store → Bundle ID + Certificate 선택 → 다운로드 → Xcode 자동 인식.

### 5. Xcode Workspace 자동 서명

```bash
cd _apps/reactNative/ShopjoyFoApp
open ios/ShopjoyFoApp.xcworkspace
```

Xcode → 타겟 → **Signing & Capabilities**:
- ✅ Automatically manage signing
- Team 선택

### 6. Push Notifications + Background Modes Capability

**+ Capability**:
- Push Notifications
- Background Modes → ✅ Remote notifications

### 7. APNs 키를 Firebase 에 업로드

[`21.구글설정하기.md`](21.구글설정하기.md) 의 **3-3단계** 참조.

### 8. App Store Connect 앱 생성

https://appstoreconnect.apple.com → 나의 앱 → **+** → 새 앱:

| 항목 | 값 |
|---|---|
| 플랫폼 | iOS |
| 이름 | ShopJoy |
| Bundle ID | `com.shopjoy.fo` |
| SKU | `SHOPJOY-FO-001` |

---

## 환경별 Bundle ID 분리

`ios/ShopjoyFoApp.xcodeproj/project.pbxproj` 또는 Xcode UI → Build Settings → `PRODUCT_BUNDLE_IDENTIFIER`:

| Configuration | Bundle ID |
|---|---|
| Debug-Local / Release-Local | `com.shopjoy.fo.local` |
| Debug-Dev / Release-Dev | `com.shopjoy.fo.dev` |
| Debug-Prod / Release-Prod | `com.shopjoy.fo` |

각 환경별 Provisioning Profile 도 별도 발급해야 자동 서명 인식.
