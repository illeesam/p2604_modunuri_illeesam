# 11. React Native 개발환경 구축 — 따라하기

## 사전 준비

- [ ] Windows 10/11 / macOS 14+ / Ubuntu 22+
- [ ] 디스크 50 GB 이상
- [ ] 인터넷

---

## 1단계: Node.js 20 LTS 설치

**Windows**:
```powershell
winget install OpenJS.NodeJS.LTS
```

**macOS**:
```bash
brew install node@20
```

확인:
```bash
node --version    # v20.x
```

---

## 2단계: JDK 17 설치 (Android)

**Windows**:
```powershell
winget install EclipseAdoptium.Temurin.17.JDK
```

**macOS**:
```bash
brew install --cask temurin@17
```

확인: `java -version`

---

## 3단계: Android Studio + SDK

https://developer.android.com/studio 다운로드 → 설치 → SDK Manager 에서:
- **Android 14 (API 34)** SDK Platform
- **Android SDK Build-Tools 34.0.0**
- **Android SDK Platform-Tools**
- **Android Emulator**
- **AVD** (Pixel 7, API 34)

환경변수:

**Windows**:
```powershell
[System.Environment]::SetEnvironmentVariable('ANDROID_HOME', "$env:LOCALAPPDATA\Android\Sdk", 'User')
$path = [System.Environment]::GetEnvironmentVariable('Path', 'User')
$path += ";$env:LOCALAPPDATA\Android\Sdk\platform-tools;$env:LOCALAPPDATA\Android\Sdk\emulator"
[System.Environment]::SetEnvironmentVariable('Path', $path, 'User')
```

**macOS** (`~/.zprofile`):
```bash
export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/platform-tools:$ANDROID_HOME/emulator
```

확인 (새 터미널):
```bash
adb --version
```

---

## 4단계: Watchman 설치 (macOS, 선택)

```bash
brew install watchman
```

> Windows 는 미사용. RN 0.71+ 부터 fb-watchman 자동 fallback.

---

## 5단계: Xcode 설치 (macOS, iOS 빌드 시만)

App Store → Xcode 설치 → `xcode-select --install` → `sudo gem install cocoapods`.

---

## 6단계: ShopJoy RN 프로젝트 의존성 설치

```bash
cd _apps/reactNative/ShopjoyFoApp
npm install
```

```
added 1500 packages in 90s
```

(많은 의존성이 있어 5~10분 소요)

iOS Pod 설치 (macOS 만):
```bash
cd ios
pod install
cd ..
```

BO 앱도:
```bash
cd ../ShopjoyBoApp
npm install
cd ios && pod install && cd ..    # macOS 만
```

---

## 7단계: 네이티브 디렉토리 생성 (없는 경우)

`android/` 또는 `ios/` 폴더가 없으면 RN init 으로 생성:

```bash
cd /tmp
npx @react-native-community/cli init ShopjoyFoApp --version 0.75.4 --skip-install
```

생성된 `android/` `ios/` 폴더만 본 프로젝트로 복사:

```bash
cp -r /tmp/ShopjoyFoApp/android _apps/reactNative/ShopjoyFoApp/
cp -r /tmp/ShopjoyFoApp/ios _apps/reactNative/ShopjoyFoApp/
```

> **주의**: 본 프로젝트의 `App.tsx` / `index.js` / `package.json` 은 절대 덮어쓰지 마세요.

---

## 8단계: react-native-rename 으로 패키지명 변경

```bash
cd _apps/reactNative/ShopjoyFoApp
npx react-native-rename "ShopjoyFoApp" -b com.shopjoy.fo
```

---

## 9단계: Metro 실행 검증

```bash
npm start
```

다음과 같이 보이면 OK:

```
Welcome to React Native!
                                                                                
- Press a │ run on Android
- Press i │ run on iOS
- Press d │ open Dev Menu
```

별도 터미널에서:
```bash
npm run android:local
```

에뮬레이터 부팅 + ShopJoy 앱 실행되면 환경 구축 완료.
